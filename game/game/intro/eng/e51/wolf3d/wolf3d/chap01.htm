1.物品
  (1)青色小方塊 = 子彈

  (3)青色小圓圈 = 餐點

  (6)暗青色g = 機槍
  (7)青色g = 鏈槍
  (8)黃色小十字、小圓圈、小方塊、n = 加分寶物
  (9)黃色k、青色k = 鑰匙
2.敵人：白色為必定出現，棕色等級3以上會出現，暗棕色等級4才會出現
  (1)G = 警衛
  (2)D = 警犬
  (3)S = SS衝鋒隊
  (4)O = 軍官
  (5)M = 變種人
  (6)* = 魔王
  (7)H = 希特勒
3.其他
  (1)青色箭頭 = 本關主角開始處
  (2)白框方格 = 暗格
  (3)黃色格子 = 過到下一關的電梯（呈品字共三個）
  (4)黃色門或青色門 = 鎖住的門，需對應的鑰匙開啟
