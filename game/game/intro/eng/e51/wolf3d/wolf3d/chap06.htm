1.物品
  (1)青色小方塊 = 子彈

  (3)青色小圓圈 = 餐點

  (1)G = 警衛
  (2)D = 警犬
  (3)S = SS衝鋒隊
  (4)O = 軍官
  (5)M = 變種人
  (6)* = 魔王
  (7)H = 希特勒

  (1)青色箭頭 = 本關主角開始處
  (2)白框方格 = 暗格
  (3)黃色格子 = 過到下一關的電梯（呈品字共三個）
  (4)黃色門或青色門 = 鎖住的門，需對應的鑰匙開啟
