密技：修改PROJECT\vcop2.ini，在[GameSetting]裡加入一行文字：

Extra=2

進入遊戲後在標題畫面按F6鍵，會看到兩個新選項：Special和Cheat：

1.Special

Single Click Reload = 滑鼠點一下即可填裝彈藥
Random Mode = 敵人隨機出現
Mirror Mode = 遊戲視窗顯現出反射的影子
Big Head Mode = 將遊戲主角變成大頭狀

2.Cheat（使用後無法輸入排行榜名字）

Auto Reload = 用完彈藥後自動填裝
Gun Select = 在遊戲進行時可以選擇想要的武器
Special Gun Select = 在遊戲進行時可以選擇特殊槍械

亦可修改vcop2.ini的Continue=#，將數字改成10，即可無限接關。

操作：

F3 = 暫停
F4 = 變更螢幕顯示方式
F5 = 變更操作方式與按鍵
Alt = 開啟系統選單
Alt+F2 = 重新開始
Alt+F4 = 結束遊戲
滑鼠左鍵 = 射擊
滑鼠右鍵 = 裝載（連按兩次，隨時可裝填）
Enter = 開始
