記憶體修改
存檔有加密，故只能修改記憶體。首先利用GameMaster搜尋金錢（MM的位置），或開始新遊
戲後搜尋（1,0,0,0,0,2,2,1,1,4），亦即金錢下FG、RG的位置，然後修改記憶體如下（遊戲
中仍可改）：

  -- -- -- -- -- H2 -- H1 MM MM MM MM 00 00 00 00
  -- -- -- -- -- -- FG RG -- LS RS GG -- -- -- SS
  XX ST -- -- -- -- -- -- -- -- -- -- -- -- FG RG
  -- LS RS GG -- -- -- SS XX ST -- -- -- -- -- --
  -- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --
  -- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --
  -- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --
  -- -- -- -- -- -- V1 V2 -- -- -- -- -- -- -- --

Tyrian 2000的記憶體格式則為：

  -- -- -- -- -- -- -- -- -- -- -- -- H2 -- H1 00
  MM MM MM MM 00 00 00 00 -- -- -- -- -- -- FG RG
  -- LS RS GG -- -- -- SS XX ST -- -- -- -- -- --
  -- -- -- -- -- -- FG RG -- LS RS GG -- -- -- SS
  XX ST -- -- -- -- -- -- -- -- -- -- -- -- -- --
  -- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --
  -- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --
  -- -- -- -- -- -- -- -- -- -- -- -- -- -- V1 V2

MM = 金錢，別改太大（參閱攻略說明）
H1 = 關卡難度（決定出現敵人數量），01=Easy、02=Normal、03=Hard、
     04=Impossible、08=Suicide／Lord of Game
H2 = 敵人強度（決定出現敵人強度），初始數值同關卡難度，每過一關系統會重新依照金錢
     總數調整，最大到9（當然還可以更高）
FG = Front Gun
RG = Rear Gun
LS = Left Sidekick
RS = Right Sidekick
GG = Generator（06h最佳）
SS = Shield（0Ah最佳，若為Tyrian2000，則0Bh最佳）
XX = 特殊武器
ST = Ship Type（0Ch最佳，若金錢要少，則0Dh最佳）
V1 = Front Gun等級，最大0Bh
V2 = Rear Gun等級，最大0Bh

注意第二組FG、RG以下是Upgrade Ship時使用的，應該修改第一組。另外，Sidekick武器剩
餘彈數在記憶體裡也是固定的，進入遊戲中找到後鎖定，便可以享受無限彈數的強力武器。
