1.Torm

販賣裝備：
  Ship Type = Gencore Phoenix
  Shield = MicroCorp LXS Class A
  Generator = Gencore Custom MR-12
  Front Gun = Protron, Missile Launcher, Mega Pulse
  Rear Gun = Starburst, Sonic Wave, Scatter Wave
  Left Sidekick = Single Shot Option, Mini-Missile, Side Ship
  Right Sidekick = 同上

武器升級：
  (1)過橫向敵機後第5群快速敵機，右邊第2隻不要打，等到後面的龍接觸再後退時才打，
     可提升Rear武器等級。





  (2)過兩旁射彈的瀑布後第3群快速敵機，會自動伴隨著提升Front武器等級的東西。

接關：
  (1)結束後，進入Gyges。

2.Gyges

販賣裝備：
  Ship Type = Gencore Phoenix
  Shield = MicroCorp LXS Class B
  Generator = Gencore Custom MR-12
  Front Gun = Multi-Cannon, Mega Cannon, Protron
  Rear Gun = Sonic Wave, Protron, Vulcan Cannon
  Left Sidekick = MegaMissile, Phoenix Device, Mint-O-Ship
  Right Sidekick = 同上

武器升級：
  (1)一開始的紅色物體打掉，可提升Rear武器等級。



接關：
  (1)打下3隻像蜘蛛縮著的東西，第1隻在爪船出現時，第2隻在加速結束不久後，在蛇怪後
     不久出現第3隻，打下最後一隻便會出現可進入Gem War隱藏關的泡泡。







  (2)未進入隱藏關者，進入Bonus。
