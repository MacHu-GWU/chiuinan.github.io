1.Tyrian

初始販賣裝備（防禦只列最好的裝備，以下皆然）：
  Ship Type = USP Talon
  Shield = Gencore High Energy Shield
  Generator = Gencore Custom MR-12
  Front Gun = Pulse-Cannon, Multi-Cannon, Vulcan Cannon, Protron
  Rear Gun = Multi-Cannon, Sonic Wave, Heavy Guided Bombs
  Left Sidekick = Plasma Storm, Mint-O-Ship, Satellite Marlo
  Right Sidekick = Plasma Storm, MicroSol FrontBlaster, Satellite Marlo

武器升級：
  (1)第一架U型敵機等它往上飛時再打下它，可提升Rear武器等級。



接關：
  (1)最右側被螢幕切一半的岩石（第二顆），打破它出現可進入Bubbles隱藏關的泡泡。



  (2)底下狀態出現"Approaching enemy platforms"後，出現圓形敵機，先殺第一隻，再殺
     最後一隻（誤擊中也不行），會出現可進入Holes隱藏關的泡泡。



  (3)一開始的U型敵機，除第一架外都不殺，則通過spike後，出現一排5架往下的敵機，後
     面會再多一架，擊落它會出現可進入Soh Jin隱藏關的泡泡。



 未進入隱藏關者，最後選擇進入Asteroid-1或Asteroid-2。
備註：
  (1)進入Bubbles增加金錢較快，Soh Jin則有較好的Ship Type、Shield和Generator。
  (2)之後選擇Asteroid-1或Asteroid-2時，建議選Asteroid-2。

5.Asteroid-1

接關：
  (1)到很後面停下清光後出現的人面石，打掉會出現可進入Asteroid-?隱藏關的泡泡。



  (2)未進入隱藏關者，進入Savara。

6.Asteroid-2

武器升級：
  (1)第一架坦克等它迴轉後再打，可得到特殊武器或提升Front武器等級。







  (2)第二群坦克中右邊移動的一部，等它迴轉後再打，可提升Rear武器等級。





接關：
  (1)過3次停下清光地方後的品字形三顆人面岩石，右下那顆打掉，會出現可進入MineMaze
     隱藏關的泡泡。

     若是Tyrian 2000，則是打上面那顆，會出現四個可進入MineMaze隱藏關的泡泡。

  (2)未進入隱藏關者，進入Savara。

7.MineMaze

販賣裝備：
  Ship Type = USP Fang
  Shield = MicroCorp LXS Class A
  Generator = Standard MicroFusion
  Front Gun = Multi-Cannon, Mega Cannon, Missile Launcher
  Rear Gun = Sonic Wave, Protron, Vulcan Cannon
  Left Sidekick = Charge Cannon, Phoenix Device, Mini-Missile
  Right Sidekick = 同上



武器升級：
  (1)過第二個閘門開始分道時，走最右邊，打掉第二個閘門，可得到特殊武器或提升Front
     武器等級。



  (2)連排草莓後的中間通道有可提升Rear武器等級的敵機。



接關：
  (1)連排草莓後的左邊通道有可進入Asteroid-?隱藏關的敵機，但和中間通道提升Rear武器
     等級的敵機幾乎同時出現。最好先打左邊敵機後，再迅速回到中間通道。



  (2)未進入隱藏關者，進入Savara。

10.Savara

販賣裝備：
  Ship Type = USP Talon
  Shield = Gencore High Energy Shield
  Generator = Standard MicroFusion
  Front Gun = Pulse-Cannon, Multi-Cannon, Missile Launcher
  Rear Gun = Protron, Vulcan Cannon, Heavy Guided Bombs
  Left Sidekick = Dual Shot Option, MegaMissile, Post-it Mine
  Right Sidekick = 同上

備註：
  (1)Hard級以上和Normal級以下使用的地圖與敵機數量／出現位置均不同。
Normal級以下武器升級：
  (1)打完第2隻氣球船後，出現5隻敵機，打掉最右邊那隻可提升Rear武器等級。



Hard級以上武器升級：
  (1)打完第3隻大型飛機後第一群快速敵機，打掉右邊第2隻可得特殊武器或提升Front武器
     等級。



  (2)後期出現綠色飛機過炮陣後第二群快速敵機，打掉最右邊第2隻可提升Rear武器等級。



接關：
  (1)結束後，進入Bonus。
