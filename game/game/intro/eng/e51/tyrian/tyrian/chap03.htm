1.Gauntlet

3.Bonus

販賣裝備：
  Ship Type = Gencore Maelstrom
  Shield = MicroCorp LXS Class B
  Generator = Gencore Custom MR-12
  Front Gun = Multi-Cannon, Mega Pulse, Heavy Missile Launcher
  Rear Gun = Starburst, Multi-Cannon, Sonic Wave
  Left Sidekick = Dual Shot Option, Atom Bombs, Phoenix Device
  Right Sidekick = 同上

武器升級：
  (1)開始的第一架敵機打掉，可提升Rear武器等級。



  (2)第一排敵機擋路後的第一隻敵機打掉，可得特殊武器或提升Front武器等級。



  (3)第二排敵機擋路後的第一隻敵機打掉，可提升Rear武器等級。



  (4)第二隻和開始第一架一樣的敵機打掉，可得特殊武器或提升Front武器等級。



接關：
  (1)結束後，進入Stargate。

4.Stargate

販賣裝備：
  Ship Type = MicroCorp Stalker
  Shield = MicroCorp HXS Class A（Tyrian 2000為Gencore Solar Shield）
  Generator = Gencore Custom MR-12
  Front Gun = Multi-Cannon, Mega Pulse, Heavy Missile Launcher
  Rear Gun = Starburst, Multi-Cannon, Sonic Wave
  Left Sidekick = Dual Shot Option, Atom Bombs, Phoenix Device
  Right Sidekick = 同上

備註：
  (1)本關有兩波飛釘，各有一個可擊碎，擊碎後分別會有2個5000元的寶石。



接關：
  (1)第二波飛釘後的第2群4泡泡（在偏右邊），打掉出現4個進入隱藏關的泡泡，其中兩個
     進入Sawblades隱藏關，兩個進入Asteroid City。



  (2)未進入隱藏關者，進入Camanis。

5.Sawblades

販賣裝備：
  Ship Type = MicroCorp Stalker-B
  Shield = MicroCorp HXS Class A（Tyrian 2000為Gencore Solar Shield）
  Generator = Advanced MicroFusion
  Front Gun = Pulse-Cannon, Zica Laser, Lightning Cannon
  Rear Gun = Fireball, Rear Heavy Missile Launcher, Rear Mega Pulse
  Left Sidekick = Charge Cannon, Atom Bombs, MicroBomb
  Right Sidekick = 同上

武器變更：
  (1)將所有胡蘿蔔都打掉，最後一隻會掉將武器轉換成HotDog的東西。



接關：
  (1)結束後，進入Camanis。

  (2)前述後面的大炮台出現的第一架敵機打掉，可提升Rear武器等級。

12.New Deliani

販賣裝備：
  Ship Type = MicroCorp Stalker-C
  Shield = MicroCorp HXS Class A（Tyrian 2000為Gencore Solar Shield）
  Generator = Gravitron Pulse-Wave
  Front Gun = Laser, Zica Laser, Lightning Cannon
  Rear Gun = Fireball, Rear Heavy Missile Launcher, Rear Mega Pulse
