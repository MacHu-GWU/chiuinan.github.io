德軍總部3D命運之矛 終極挑戰任務包 圖文攻略（作者－青衫）
*** 攻略圖示說明（地圖由MapEdit 8.1產生）

1.物品
  (1)青色小方塊 = 子彈、子彈箱
  (2)暗青色小方塊 = 狗食
  (3)青色小圓圈 = 餐點
  (4)青色小十字 = 急救包
  (5)青色大圓圈 = 加命
  (6)暗青色g = 機槍
  (7)青色g = 鏈槍
  (8)黃色小十字、小圓圈、小方塊、n = 加分寶物
  (9)黃色k、青色k = 鑰匙
2.敵人：白色為必定出現，棕色等級3以上會出現，暗棕色等級4才會出現
  (1)G = 警衛
  (2)D = 警犬
  (3)S = SS衝鋒隊
  (4)O = 軍官
  (5)M = 變種人
  (6)H = 鬼魂
  (7)* = 魔王
3.其他
  (1)青色箭頭 = 本關主角開始處
  (2)白框方格 = 暗格
  (3)黃色格子 = 過到下一關的電梯
  (4)黃色門或青色門 = 鎖住的門，需對應的鑰匙開啟

*** Ultimate Challenge（終極挑戰）

第3-1關 標準時間1:30







第3-2關 標準時間3:30




先往下到西南拿青鑰匙，再到中央拿黃鑰匙，然後北上過關


第3-3關 標準時間2:45




先到西南拿青鑰匙，再到東北拿黃鑰匙，最後到東南過關


第3-4關 標準時間3:30




1.進入第3-19秘密關
2.進入第3-5關

若想快速過關，直接進入南方電梯即可進入第3-5關

若想進入秘密關，就必須拿取青鑰匙和黃鑰匙，再進入西南的電梯

最西邊暗格要注意推動方向，避免卡住通道


第3-19秘密關




1.進入第3-5關
先到西北拿青鑰匙，再到東北拿黃鑰匙


第3-5關魔王關




先一路到東北拿青鑰匙

本關魔王為一手鏈槍一手機槍的Submarine Willy，打死他可撿取黃鑰匙


第3-6關 標準時間4:30




一路到東南拿黃鑰匙，即可到西北過關
西南青鑰匙只是用來拿寶物用


第3-7關 標準時間3:15




先到東南拿黃鑰匙，即可到西北過關

西北青鑰匙只是用來拿寶物用，兩個電梯都到第3-8關


第3-8關 標準時間2:45




先到西南拿青鑰匙，再到東北拿黃鑰匙，一路繞到中央電梯過關


第3-9關 標準時間4:45




先到西北拿青鑰匙，再到東北拿黃鑰匙，然後到東南電梯過關


第3-10關魔王關




本關魔王為一手機槍一手火箭炮的Prof. Quarkblitz，打死他可撿取黃鑰匙


第3-11關 標準時間6:30




先從西邊繞到中央拿青鑰匙，再到西南拿黃鑰匙，然後到北邊電梯過關


第3-12關 標準時間4:30




1.進入第3-20秘密關
2.進入第3-13關

若想快速過關，直接到東南電梯即可進入第3-13關

但若想進入秘密關，便必須拿青鑰匙、再拿黃鑰匙，然後進入東北電梯

東南的暗格均只有一種推法，不會卡死，可以放心儘量推


第3-20秘密關




1.進入第3-13關
本關沒有捷徑，必須繞完整圈並依次拿取青黃鑰匙後，才能到達出口


第3-13關 標準時間2:45




先往東南到西南拿青鑰匙，再到西北拿黃鑰匙，然後到中央電梯過關


第3-14關 標準時間4:30




先往北拿黃鑰匙，再往南拿青鑰匙，然後從西南一路到西北電梯過關


第3-15關 標準時間6:00




先繞到中央依次拿取黃鑰匙、青鑰匙，再到西南電梯過關


第3-16關魔王關




魔王為手持2鏈槍的Hans Von Schlieffen，殺死可拿取黃鑰匙

青鑰匙是用來拿寶物的，與過關無關

西北秘道必須持續推動暗格才能打通


第3-17關 標準時間6:00




先往南到西南拿青鑰匙，即可從西邊到電梯過關

如果嫌雲霧遮住視線不好戰鬥，便回頭到東邊拿黃鑰匙，再從西北進入電梯



第3-18關魔王關（傳送關）




1.傳送至第3-21關（關卡不視為結束，左下關卡編號仍為18）

魔王為2鏈槍2火箭炮的機器人（Robot），殺死可拿取黃鑰匙


第3-21關魔王關




鬼魂是無法消滅的，殺死後會在幾秒之後原地復活

魔王為Devil Incarnate，殺死它即結束
