遊戲密技

密技：

1.以SPEAR -DEBUGMODE啟動遊戲。
2.進入遊戲後（不是主選單），同時按下Shift+Alt+Backspace鍵啟動密技。
3.密技啟動後，隨時可按下：
    Tab+I = 增加分數、健康、子彈、武器
    Tab+G = 無敵
    Tab+H = 傷害自己
    Tab+V = 增減遊戲速度
    Tab+S = 慢動作
    Tab+E = 跳到下一關
    Tab+W = 跳到某關
    Tab+C = 顯示本關統計資料
    Tab+F = 顯示位置資訊
    Tab+M = 顯示記憶體使用狀況
    Tab+B = 改變邊框顏色
    Tab+Q = 離開遊戲，回到Dos
