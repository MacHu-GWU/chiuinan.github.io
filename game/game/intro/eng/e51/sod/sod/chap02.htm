德軍總部3D命運之矛 回歸危險任務包 圖文攻略（作者－青衫）

*** 攻略圖示說明（地圖由MapEdit 8.1產生）

1.物品
  (1)青色小方塊 = 子彈、子彈箱
  (2)暗青色小方塊 = 狗食
  (3)青色小圓圈 = 餐點
  (4)青色小十字 = 急救包
  (5)青色大圓圈 = 加命
  (6)暗青色g = 機槍
  (7)青色g = 鏈槍
  (8)黃色小十字、小圓圈、小方塊、n = 加分寶物
  (9)黃色k、青色k = 鑰匙
2.敵人：白色為必定出現，棕色等級3以上會出現，暗棕色等級4才會出現
  (1)G = 警衛
  (2)D = 警犬
  (3)S = SS衝鋒隊
  (4)O = 軍官
  (5)M = 變種人
  (6)H = 鬼魂
  (7)* = 魔王
3.其他
  (1)青色箭頭 = 本關主角開始處
  (2)白框方格 = 暗格
  (3)黃色格子 = 過到下一關的電梯
  (4)黃色門或青色門 = 鎖住的門，需對應的鑰匙開啟

*** Return to Danger（回歸危險）

第2-1關 標準時間1:30




東北暗格要注意推的方向，否則會壓到寶物無法拿取

本關無鑰匙，所以西北的鏈槍無法用正常方式獲得

唯一方式是引敵人開門，然後在門前殺死他，讓門無法關閉（很困難）


第2-2關 標準時間3:30







第2-3關 標準時間2:45




中央偏西南的暗格要推正確方向，否則會堵死通道


第2-4關 標準時間3:30




1.進入第2-19秘密關
2.進入第2-5關

西北4暗格處要由南到北推動

西邊3暗格處要正確推動，否則會擋住通道



第2-19秘密關




1.進入第2-5關
先到東南拿青鑰匙，再經密道到西南拿黃鑰匙


第2-5關魔王關




先從東南到東北拿青鑰匙

西北5暗格要往東推

本關魔王為一手鏈槍一手機槍的Submarine Willy，打死他可撿取黃鑰匙


第2-6關 標準時間4:30







第2-7關 標準時間3:15




若想快速過關，直接往南進電梯即可

若要撿物品，便先進南邊中央兩旁房間拿鑰匙


第2-8關 標準時間2:45




若想快速過關，直接到東南進電梯即可

不過本關+5000分物品頗多，想高分最好還是去拿取


第2-9關 標準時間4:45




若想快速過關，直接走進電梯即可


第2-10關魔王關




先拿最西南的青鑰匙，西南的黃鑰匙都是拿不到的

本關魔王為一手機槍一手火箭炮的Prof. Quarkblitz，打死他可撿取黃鑰匙

注意不要在物品堆裡打死，否則黃鑰匙可能會掉不出來（消失），造成卡關


第2-11關 標準時間6:30




若想快速過關，直接到東南進電梯即可


第2-12關 標準時間4:30




1.進入第2-20秘密關
2.進入第2-13關

若想快速過關，直接經秘道到南方電梯即可進入第2-13關

但若想進入秘密關，便必須通過東南暗格迷宮

東南的暗格均只有一種推法，不會卡死，可以放心儘量推


第2-20秘密關




1.進入第2-13關
本關沒有捷徑，必須繞完整圈後才能到達出口


第2-13關 標準時間2:45







第2-14關 標準時間4:30




西邊有些虛暗格，必須將最南的實格往北逐一推動佔據，以達成100%

有些暗格可能卡住通道，但沒有影響

若想快速過關，直接走進電梯即可

不過本關+5000分物品頗多，想高分最好還是去拿取


第2-15關 標準時間6:00




本關敵人多又集中，開門後別馬上進入，逐一引出解決


第2-16關魔王關




魔王為手持2鏈槍的Hans Von Schlieffen，殺死可拿取黃鑰匙


第2-17關 標準時間6:00




東邊暗格要注意推動方向，不要壓到黃鑰匙


第2-18關魔王關（傳送關）




1.傳送至第2-21關（關卡不視為結束，左下關卡編號仍為18）

北邊暗格要注意推動方向，別卡住通道

魔王為2鏈槍2火箭炮的機器人（Robot），殺死可拿取黃鑰匙


第2-21關魔王關




鬼魂是無法消滅的，殺死後會在幾秒之後原地復活

魔王為Devil Incarnate，殺死它即結束
