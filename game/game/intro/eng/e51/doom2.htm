名稱：毀滅戰士2人間地獄 (Doom 2 - Hell on Earth，英文版)
簡介：Id Software 1994年出品的第一人稱射擊遊戲，劇情承接第一代，玩法
      亦相同。本代增加了一件武器、十種敵人，雖然只有一個章節，但關卡
      數有32個，比一代三個章節的關卡數還多。儘管內容和一代相差不多，
      仍然獲得1994年最佳科幻遊戲獎，也是值得推薦的一款遊戲。
