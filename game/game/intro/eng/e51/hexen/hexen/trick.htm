遊戲密技

密技：

在遊戲中輸入以下密碼：

SATAN = 無敵模式（過高摔下仍會死亡）
NRA = 補滿所有武器、法力、護甲
CONAN = 移除所有武器
CLUBMED = 補滿生命
LOCKSMITH = 補滿鑰匙
INDIANA = 獲得所有寶品
SHERLOCK = 獲得所有過關物品
CASPER = 穿牆模式
BUTCHER = 殺死本關所有怪物
SHADOWCASTERx = 變換職業，0=戰士，1=牧師，2=法師
VISITxy = 跳關，xy=關卡編號（不是地圖編號）
INIT = 本關卡重新進行
DELIVERANCE = 把自己變成豬
MARTEK = 輸入三次後自殺
WHERE = 顯示所在位置座標
PUKExy = 執行內定的Script，xy = Script編號
NOISE = 音效除錯訊息
~xy = 選擇CD音軌，xy = 編號編號

在地圖模式中輸入以下密碼：

MAPSCO = 第一次秀出全部地圖、第二次秀出怪物位置、第三次則切回正常模式
