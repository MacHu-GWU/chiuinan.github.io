3.修正暗格2格移動變3格移動的問題
  00 01 76 08
  FF 00 -- --

  從DosBox觀察，系統會在用時>100h時停止暗格移動，而每移一格為80h，當移到第二格時
  剛好為100h。由於<=100h，因此還會繼續移動。只需將比對值減一便可以只移2格。如果
  還是移動3格，便將比對的時間值再減少一些，如中間值C0h會更保險些。

MAPHEAD.xxx：關卡地圖指標檔

  RLEWtag：2 byte整數（RLEW解壓用）
  關卡地圖指標：4 byte整數，共100個，指向地圖資訊檔，<0表不存在，0表結束

1.關卡地圖資訊（由關卡地圖指標檔指向）
  地圖壓縮資料指標：3個4 byte整數，指向地圖資訊檔
  地圖壓縮資料大小：3個2 byte整數
  地圖水平大小：2 byte整數，固定40h=64
  地圖垂直大小：2 byte整數，固定40h=64
  地圖名稱：16 byte
2.地圖壓縮資料（由關卡地圖資訊指標指向）：須按後述方式解壓

地圖資料共有3組，內容為資料編號：

1.地圖背景資料（MapData），例如牆壁、電梯等。
2.地圖物件資料（ObjData），例如寶物、鑰匙、桌椅等。
3.暗格資料

各資料編號均有固定意義（寫死在程式中），請自行參閱地圖編輯器MapEdit所附的MapData.*
和ObjData.*。秘密關的關卡連結等，亦寫死在程式中，外界無法變更。

*** 地圖資料解壓法

註：原始資料大小 = 地圖水平大小 * 地圖垂直大小 * 2

dest_len = 地圖壓縮資料開頭的2 byte整數
Carmack解壓(地圖壓縮資料+2,dest,dest_len)
地圖原始資料大小 = dest開頭的2 byte整數
RLEW解壓(dest+2,地圖原始資料,地圖原始資料大小)

*** Carmack解壓(source,dest,length)

while (輸出緩衝區尚未填滿)
    ch = 讀取2 byte整數
    chhigh = ch高位元組
    if (chhigh == 0xA7) // NEARTAG //
        count = ch低位元組
        if (count == 0)
            讀取1 byte做為ch的低位元組
            輸出ch (2 byte整數)
        else
            offset = 讀取2 byte整數
            複製輸出緩衝區目前位置往前offset*2的位置共count個2 byte整數
        end if
    else if (chhigh == 0xA8) // FARTAG //
        count = ch低位元組
        if (count == 0)
            讀取1 byte做為ch的低位元組
            輸出ch (2 byte整數)
        else
            offset = 讀取2 byte整數
            複製輸出緩衝區開始往後offset*2的位置共count個2 byte整數
        end if
    else
        輸出ch (2 byte整數)
    end if

*** RLEW解壓(source,dest,length)

while (輸出緩衝區尚未填滿)
    value = 讀取2 byte整數
    if (value != RLEWtag)
        輸出value (2 byte整數)
    else
        count = 讀取2 byte整數
        value = 讀取2 byte整數
        輸出count次value (2 byte整數)
    end if

攻略：（作者－青衫）

1.初始武器=刀、槍，子彈（Ammo）=8，健康（Health）=100%，生命（Lives）=3，分數
  （Score）=0，所在關卡（Floor）=1。
2.被敵人擊中時健康度下降，下降至0%時即死亡，減少一命並從本關重新開始。
3.殺敵或撿拾寶物可獲得分數，每40000分增加一命，最多9命。
4.經過物品時會自動撿拾。除了地上可見物品外，殺死警衛經過屍體時也可撿拾子彈，最多
  99發（SS衝鋒隊有時會掉機槍）。
5.某些門需要鑰匙才能開啟，撿到鑰匙會顯示在子彈的右邊2小格內。
6.每過一關（無法回頭），依照各項指標給予額外分數：
    (1)殺死敵人100%：10000分。
    (2)拿取寶物100%：10000分。
    (3)推動暗格100%：10000分。
    (4)低於標準時間：每秒500分。
  秘密關或魔王關不以上述指標為標準，過關直接加15000分。
7.敵人不會站著不動，聽到槍聲甚至會四處搜尋或開門出來攻擊。

第3-5關魔王關




先一路到東北拿青鑰匙

本關魔王為一手鏈槍一手機槍的Submarine Willy，打死他可撿取黃鑰匙


第3-6關 標準時間4:30




一路到東南拿黃鑰匙，即可到西北過關
西南青鑰匙只是用來拿寶物用


第3-7關 標準時間3:15




先到東南拿黃鑰匙，即可到西北過關

西北青鑰匙只是用來拿寶物用，兩個電梯都到第3-8關


第3-8關 標準時間2:45




先到西南拿青鑰匙，再到東北拿黃鑰匙，一路繞到中央電梯過關


第3-9關 標準時間4:45




先到西北拿青鑰匙，再到東北拿黃鑰匙，然後到東南電梯過關


第3-10關魔王關




本關魔王為一手機槍一手火箭炮的Prof. Quarkblitz，打死他可撿取黃鑰匙


第3-11關 標準時間6:30




先從西邊繞到中央拿青鑰匙，再到西南拿黃鑰匙，然後到北邊電梯過關


第3-12關 標準時間4:30




1.進入第3-20秘密關
2.進入第3-13關

若想快速過關，直接到東南電梯即可進入第3-13關

但若想進入秘密關，便必須拿青鑰匙、再拿黃鑰匙，然後進入東北電梯

東南的暗格均只有一種推法，不會卡死，可以放心儘量推


第3-20秘密關




1.進入第3-13關
本關沒有捷徑，必須繞完整圈並依次拿取青黃鑰匙後，才能到達出口


第3-13關 標準時間2:45




先往東南到西南拿青鑰匙，再到西北拿黃鑰匙，然後到中央電梯過關


第3-14關 標準時間4:30




先往北拿黃鑰匙，再往南拿青鑰匙，然後從西南一路到西北電梯過關


第3-15關 標準時間6:00




先繞到中央依次拿取黃鑰匙、青鑰匙，再到西南電梯過關


第3-16關魔王關




魔王為手持2鏈槍的Hans Von Schlieffen，殺死可拿取黃鑰匙

青鑰匙是用來拿寶物的，與過關無關

西北秘道必須持續推動暗格才能打通


第3-17關 標準時間6:00




先往南到西南拿青鑰匙，即可從西邊到電梯過關

如果嫌雲霧遮住視線不好戰鬥，便回頭到東邊拿黃鑰匙，再從西北進入電梯
