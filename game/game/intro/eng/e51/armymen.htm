名稱：玩具奇兵1／玩具兵大戰1 (Army Men，英文版)
簡介：3DO 1998年出品的第三人稱射擊戰術遊戲 [待補]。



保護：光碟
操作：

數字盤上鍵 = 前進
數字盤下鍵 = 後退
數字盤上下鍵 = 前進／後退
數字盤左右鍵／ZX／滑鼠左右移 = 左右轉（+瞄準）
Space／滑鼠左鍵 = 射擊
Pad-7/9 = 翻滾
Pad-5 = 跪姿／臥姿
/ = 臥姿
1 = 主武器
2 = 重武器
3 = 特殊物品
F4/E = 進入車子
F8/D = 丟棄物品
F11/O = 顯示任務目標
F12/V = 檢視地圖
G = 變更準星形態
Enter = 滾動檢視戰場開關

友軍命令：

F1 = 跟隨
F2 = 攻擊
F3 = 防禦
F5/S = 空中打擊
F6/P = 空降部隊
F7/R = 空中偵察
F9/A = 立即攻擊
F10/F = 立即防禦
Tab = 下個單位
Backspace = 交談
