名稱：毀滅巫師2／異教徒2 (Heretic 2，英文版)
簡介：Raven Software於1998年開發的第一人稱射擊遊戲，由Activision發行 [待補]。



保護：光碟
備註：如果開新遊戲會當，可先開Tutorial或載入存檔後，再開其他新遊戲。或是替換更新
      版的quake2.dll。
操作：進入遊戲後，可變更Video settings到較為適合的解析度，亦可視窗模式。
檔案：ht2_ep_v1-06.rar = 官方1.06更新檔
      HereticIIEnhancementPackV1.07.rar = 非官方1.07更新檔
      quake2.rar = 更新版quake2.dll（開新遊戲不當機）
密技：按下~鍵，再輸入下列密技字串：

playbetter          無敵
suckitdown health   恢復生命至最大值
suckitdown mana     恢復法力至最大值
suckitdown lungs    重置待在水中的時間
suckitdown weapons	取得所有武器與攻擊法術（不包含彈藥）
suckitdown staff    變更原始武器等級（原始=1,冰=2,火=3循環）
suckitdown defences	取得所有防禦法術
suckitdown armor    取得護甲（第一次銀護甲，第二次金護甲）
suckitdown all      取得所有武器彈藥、法術與護甲，並恢復生命法力至最大值
suckitdown powerup  進入使用Tome of Power狀態
twoweeks            切換Tome of Power狀態
meatwagon           殺死所有怪物（魔王除外）
victor              無怪物模式開關
angermonsters       讓所有怪物都生氣，會前來攻擊，有時效性
crazymonsters       讓所有怪物都瘋狂，會亂跑不會攻擊，有時效性
kiwi                穿牆模式開關
showcoords          顯示座標
suckitdown chicken  變成小雞開關
kill	            自殺
weapprev            更換上一個武器（同J）
weapnext            更換下一個武器（同K）
freezeworld	        暫停遊戲
exit	            結束遊戲
quit	            結束遊戲
screenshot	        截圖

操作：

上卜鍵 = 前進／後退
Shift+上下鍵 = 前跑／後翻
左右鍵 = 左轉／右轉
Alt+左右鍵 = 左移／右移
,. = 左移／右移
Z或M = 蹲低
A或/ = 跳起
左Ctrl = 攻擊
Space = 動作
J/K = 更換上／下一個武器
    1～9 = 選擇武器
[] = 更換上／下一個防禦法術
    Q = Tome of Power
    W = Ring of Repulsion
    Y = Lightning Shield
    T = Teleport
    R = Morph Ovum
    E = Meteor Swarm
    無 = Whirlwind
Enter = 施展防禦法術
; = 立即180度轉身
Del/PgDn = 往下／往上移動視角
End = 恢復內定視角
P = 查看大地圖
U = 查看城市地圖
I = 查看關卡物品
O = 查看一般物品
S = 送出對話
F1 = 輔助說明
F5 = 快速存檔
F6 = 快速讀檔
F9 = 不鎖定鏡頭（鏡頭跟隨主角）
F10 = 鎖定鏡頭
F12 = 結束遊戲
~ = 命令視窗
+/- = 增減可見視窗大小
