名稱：馬鈴薯大兵 (Militato，英文版)
簡介：聖教士2000年製作的第三人稱3D射擊遊戲，由華彩代理發行 [待補]。



保護：光碟
備註：目前測試各式虛擬機+Win98或XP都無法執行，只能在Win64搭配dgVoodoo或DDrawCompat
      （DxWnd不行），不過使用DDrawCompat會無法截圖。
密技：在遊戲中按下~鍵，即輸入下列密技字串：

GIVE MPJACKET = 增加五件防彈背心
GIVE MPRADIO = 增加五個收音機
GIVE MPRADAR = 增加五個雷達
GIVE MPPROPELLER = 增加五個蜻蜓帽
GIVE MPPENGUIN = 增加五個企鵝炸彈
GIVE MPMINE = 增加五個地雷
GIVE MPTRAP = 增加五個陷阱
GIVE MPGRENADE = 增加五顆手榴彈
GIVE MPCLAYMORE = 增加五個定時器
GIVE MPKEY = 增加五把鑰匙
GIVE MPENERGY = 恢復體力至99點
GIVE MPGOD = 無敵
MPSKIP = 跳到下一關
1～9 = 跳到第1～9關
~ = 關閉輸入視窗

操作：

Shift+移動鍵 = 快跑
Q = 踢動箱子
A = 跳躍

說明：

1.木箱可射破或踢動後破裂，通常會有物品。某些木箱破裂後，會使得某些擋路的箱子消失
  或出現。
2.某些鑰匙箱在有鑰匙後，才可踢破。
3.踢動鐵箱會移動不會破裂。被鐵箱撞到，不管敵我都立死。
