遊戲密技

密技：

在遊戲中輸入以下密碼：

IDDQD = 無敵模式
IDKFA = 補滿各種武器、彈藥、護甲和鑰匙
IDFA = 補滿各種武器、彈藥、護甲，不含鑰匙（1.4版以後才有）
IDCHOPPERS = 獲得一把電鋸
IDSPISPOPD = 穿牆模式
IDCLEV = 跳關，後面先輸入第幾章，再輸入第幾關
IDBEHOLD = 依其後參數獲得強化裝置
             V 無敵
             S 狂暴
             I 隱形
             R 幅射防護衣
             A 地圖
             L 夜視鏡
IDMYPOS = 顯示玩家目前位置（16進位）

在地圖模式中輸入以下密碼：

IDDT = 第一次秀出全部地圖、第二次秀出怪物位置、第三次則切回正常模式
