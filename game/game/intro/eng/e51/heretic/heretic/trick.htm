遊戲密技
密技：

在遊戲中輸入以下密碼：

QUICKEN = 無敵模式
RAMBO = 補滿各種武器、彈藥（含背包）、護甲
SHAZAM = 強化武器威力開關
PONCE = 補滿生命
SKEL = 補滿鑰匙
KITTY = 穿牆模式
MASSACRE = 殺死本關所有怪物
ENGAGExy = 跳關，x=第幾章，y=第幾關
GIMMExy = 獲得寶物y個，x可為下列字母：
            a = 無敵戒指（Ring of Invincibility）
            b = 隱身魔球（Shadow Sphere）
            c = 補血藥瓶（Quartz Flask）
            d = 神秘骨灰（Mystic Urn），共享版禁止
            e = 能量之書（Tomb of Power）
            f = 火把（Torch）
            g = 定時炸彈（Time bomb of the Ancients）
            h = 化形卵（Morph Ovum）
            i = 飛翔之翼（Wings of Wrath）
            j = 混沌裝置（Chaos Device），共享版禁止
GIMMEz0 = 獲得所有寶物並填滿16個
COCKADOODLEDOO = 把自己變成小雞

在地圖模式中輸入以下密碼：

RAVMAP = 第一次秀出全部地圖、第二次秀出怪物位置、第三次則切回正常模式
