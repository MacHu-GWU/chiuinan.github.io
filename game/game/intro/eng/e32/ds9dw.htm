備註：遊戲目前測試結果只能在86Box或PCem+Win98系統上執行（但速度略慢），其餘均不行。
      Win64系統必須以下列方式修正後方能執行（虛擬機+XP仍無法執行）：

1.進行DDraw呼叫修正。
2.搭配dgVoodoo2。
3.將螢幕解析度設成1280x960後再執行，否則滑鼠游標會卡死。可利用QRes進行設定，例如
  使用下列批次檔執行：

set __COMPAT_LAYER=WIN98 HIGHDPIAWARE
QRes.exe /x:1280 /y:960
start ds9dw.exe
QRes.exe /x:1920 /y:1080

檔案：ds9dw1to4.rar = 1.01至1.04更新檔
      dwpatch5.rar = 1.04至1.05更新檔
      DDrawPatch.rar = DDraw呼叫修正（已免光碟，需先更新至1.04或1.05）
         （將程式裡的DDRAW.DLL字串改成DDRA2.DLL，1.01版可比照辦理）
      QRes.rar = QRes執行工具（內含本遊戲啟動批次檔）
