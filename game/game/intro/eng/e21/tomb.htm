名稱：古墓奇兵1／古墓麗影1 (Tomb Raider，英文版)
簡介：Core Design 1996年出品的動作冒險解謎遊戲，由Eidos代理發行，台灣由第三波代理
      發行。1998年推出獨立資料片「未完成的任務（Unfinished Business）」，並合併為
      黃金版 [待補]。



保護：光碟
檔案：tombub.rar = 獨立資料片「未完成的任務」
操作：

上鍵 = 往前跑／往下（Shift或Pad1+上鍵 = 往前走）
下鍵 = 往後跳／往上（Shift或Pad1+下鍵 = 往後走）
左右鍵 = 左右轉
Del/PgDn或Pad7/Pad9 = 左右移
End/Pad5 = 前滾翻
Alt/Pad+ = 跳躍／游泳
Space/Pad3 = 裝備武器
Ctrl/Pad-Enter = 動作（撿取等）／射擊（按著可連發）
Esc/Pad. = 物品欄／功能選項
Pad0 = 查看
