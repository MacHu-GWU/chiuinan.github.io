名稱：陰陽幻境 (Of Light and Darkness: The Prophecy，英文版)
簡介：Tribal Dreams 1998年出品的冒險解謎遊戲，由英特衛代理發行 [待補]。



保護：光碟（玩時要放入第二片，否則無法點擊New Game或Free Tour）
備註：XP以上系統安裝時，要將Setup.exe設成Win9x相容。
操作：

滑鼠左鍵 = 選擇／動作
滑鼠右鍵 = 物品欄
Space/滑鼠右鍵+左鍵 = 使用選擇的物件
Pause = 暫停
上下鍵 = 放大縮小顯示視窗
+/- = 增加減少旋轉速度
PgUp/PgDn = 控制螢幕亮度
Insert = 截圖
Esc = 回到主選單／回到遊戲
