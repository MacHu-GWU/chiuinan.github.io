名稱：古墓奇兵2：西安匕首／古墓麗影2 (Tomb Raider 2，英文版)
簡介：Core Design 1997年出品的動作冒險解謎遊戲，由Eidos代理發行，台灣由第三波代理
      發行。1999年推出獨立資料片「黃金面具（The Golden Mask）」，並合併為黃金版 [待補]。



保護：光碟
操作：

上鍵 = 往前跑／往下（Shift或Pad1+上鍵 = 往前走）
下鍵 = 往後跳／往上（Shift或Pad1+下鍵 = 往後走）
左右鍵 = 左右轉
Del、PgDn或Pad7、Pad9 = 左右移
End或Pad5 = 前滾翻
Alt或Pad+ = 跳躍／游泳
/或Pad- = 點火照明
Space或Pad3 = 裝備武器
Ctrl或Pad-Enter = 動作（撿取等）／射擊（按著可連發）
Esc或Pad. = 物品欄／功能選項
Pad0 = 查看
