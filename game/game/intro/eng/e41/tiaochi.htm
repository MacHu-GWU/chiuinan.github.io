名稱：跳棋 (Tiao Chi: The Chinese Checker Champion, 英文版)
簡介：Rick Borgen 1983年撰寫的跳棋遊戲，完全符合實體跳棋遊戲的規範。亦即玩的人可
      為2～6人，3人以下時，每人15子；4人以上時，每人10子。各個人物均可由人控制，
      或由電腦控制，是個一到多人都可以玩的遊戲，除了操作略為不便外，娛樂性還不錯。
 
保護：無。
說明：

一、新遊戲選項

1.遊戲一開始會問有多少角色一起玩（包括電腦操控的），可輸入2～6。
  How Many Players? (2 to 6):
2.接著輸入每個角色的名稱，輸入一個字母c，表示該角色由電腦操控。
3.接著輸入電腦操控AI的層級，1表新手，2表專家，3表大師（下棋時可變換）。
  Select Level of Play(1-3):
4.最後選擇每個角色的代表圖案或數字，p表圖案，n表數字。
  What type of Tokens?(p or n):
5.前面選擇圖案後，會再選擇各角色代表的顏色與圖案。

二、遊戲操控方式

1.下棋時，隨時可按下Esc鍵出現系統選單：
  (F1) Set to Novice：設定電腦操控AI的層級為1新手。
  (F2) Set to Expert：設定電腦操控AI的層級為2專家。
  (F3) Set to Master：設定電腦操控AI的層級為3大師。
  (F4) Turn Sound On/Off：音效開關。
  (F7) Start a New Game：開新遊戲。
  (F8) Take Back Move：悔棋。
  (F9) Quit This Game：結束遊戲。
  (Esc) Cont. With Game：繼續下棋。
2.按左右鍵可變換要選擇的子，決定後按下Enter鍵。此時會出現選擇的子可走的地點，按
  左右鍵可變換可走的地點，最後按下Enter鍵走棋，或換到該子時按下Enter鍵重新選擇。
