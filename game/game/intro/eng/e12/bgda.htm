名稱：柏德之門：黑暗聯盟 (Baldur's Gate: Dark Alliance，英文版)
簡介：Snowblind 2001出品的角色扮演遊戲，以3D畫面呈現，採單人即時方式戰鬥。本遊戲
      直到2021年才移植到PC，且未支援中文版 [待補]。



保護：Steam
操作：可在一開始主畫面的[Option]-[Controls]設定：

WS = 上下移動
AD = 左右移動
滑鼠右鍵 = 移動
滑鼠左鍵 = 攻擊
Shift+滑鼠左鍵 = 施法
上/下鍵 = 切換法術
左/右鍵 = 切換武器
1 = 喝補充生命藥水
2 = 喝補充法力藥水
X = 動作(交談、開箱等)
Z = 跳躍
Space = 阻擋/防禦
Tab = 自動地圖
R/T = 旋轉視角
I = 裝備選單
Enter = 確認
Esc = 取消/系統選單

說明：必須特定地點才能存檔。
