名稱：暗黑破壞神 (Diablo，英文版)
簡介：Blizzard 1996年出品的角色扮演遊戲，以斜角3D畫面呈現，採單人即時方式戰鬥。
      1997年Sierra開發了地獄火資料片，由於並非官方資料片，故無法連上官網進行網路
      遊戲 [待補]。





備註：本版為原始版1.09版+地獄火1.01版
保護：光碟，破解碼（地獄火資料片亦同）：
      Storm.dll
        74 0B 66 81 F9 05
        EB -- -- -- -- --
      但必須進行下列registry註冊（假設DIABDAT.MPQ是放在C:\Diablo）：

[HKEY_LOCAL_MACHINE\SOFTWARE\Blizzard Entertainment\Archives]
"DiabloCD"="C:\\Diablo\\"

[HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Blizzard Entertainment\Archives]
"DiabloCD"="C:\\Diablo\\"

備註：執行時若發生written的應用程式錯誤，可到[我的電腦]按右鍵選內容，到[進階]－
      [效能]－[資料執行防止]，將執行檔加入即可。
備註：地獄火資料片可修改command.txt，換成下列字串，即可多出吟遊詩人和野蠻人等兩種
      隱藏職業可選擇：

cowquest;theoquest;bardtest;multitest;barbariantest;

備註：原版與地獄火資料片的存檔並不相容，必須擇一執行。

操作：

滑鼠左鍵 = 選擇／移動／攻擊
滑鼠右鍵 = 技能／法術
P = 暫停
TAB = 地圖
Esc = 系統選單
C = 人物資料
I = 物品欄
B = 魔法書
Q = 任務日誌
S = 速檢
1-8 = 使用腰帶物品
