名稱：上古卷軸：紅色守衛 (The Elder Scrolls Adventures: Redguard，英文版)
簡介：Bethesda 1998年出品的動作冒險遊戲 [待補]。



保護：無，放入光碟才有背景音樂和動畫
備註：本遊戲必須在Windows上安裝，只能在Win9x DOS視窗或DosBox上執行。
備註：在DosBox上執行時，要將memsize設定為64，否則可能會記憶體不足而無法啟動。
操作：

上鍵 = 跑步前進
Shift+上鍵 = 走路前進
下鍵 = 後退
左右鍵 = 轉動
Shift+左右鍵 = 平移
Ctrl = 動作（攻擊／使用／交談／選擇）
Space = 跳躍
Alt = 檢視
S = 裝備武器
H = 治療生命（喝藥水）
I = 物品欄
,. = 更換上／下一個選定物品
M = 地圖
L = 記錄
Esc = 系統選單
