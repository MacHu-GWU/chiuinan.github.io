名稱：小鬼當家1 (Home Alone，英文版)
簡介：Manley & Associates 1991年出品的動作遊戲，具有解謎遊戲的元素，由Capstone代
      理發行，台灣由智冠代理發行。玩家必須在8:00到9:00時間內設置陷阱（實際大約５
      分鐘左右），以便對入侵的竊賊進行傷害。竊賊中陷阱十次即無法行動，最終目標是
      讓兩名竊賊全部喪失行為能力，因此至少要設置20個以上的陷阱，同時也要現身引誘
      竊賊中陷阱 [待補]。
 
備註：本版為1991/10/17版（智冠版為1991/08/29版）
保護：字母密碼，破解碼：
      HA.EXE (先用UNP解壓)
      (1991/08/29版)
      3B C2 74 0D 1E (共4個)
      -- -- EB -- --
      8B D0 81 F2 FF 00 88 16 08 66 81 F2 FF 00 EB 55
      8A 16 08 66 B6 -- 81 F2 FF 00 3B C2 EB 57 83 FE
      或
      (1991/10/17版)
      B4 00 8A 16 08 66 B6 00 81 F2 FF 00 3B C2 75 0E
      89 C2 80 F2 FF 88 16 08 66 80 F2 FF -- -- -- --
操作：

F1 = 撿起物品（有時需要配合跳躍鍵）
F2 = 選擇物品
F3 = 放下物品或佈置陷阱
Esc = 系統選單
