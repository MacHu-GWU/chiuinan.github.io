名稱：名車大賽 (Test Drive，英文版)
簡介：Accolade 1987年出品的飆車模擬遊戲，此系列一直流行到三代都非常受歡迎，但到了
      四代以後逐漸走下坡，而被EA的極速快感系列所取代。遊戲的目的，是儘量以最快速
      度飆完一段路，在到達終點時會有一段評語，如〞飛得太低〞等等，蠻有趣的。不過
      若速度太慢便會自動結束，否則便繼續下段路線。愈後面的路段車子及彎道愈多，而
      要求的平均速度也愈高。中間有時也會出現警車來個超速攔截，如果速度無法超過時
      速110公里的話，便會被攔下來開罰單，損失時間。不過每個路段出現的車子及彎道位
      置都是固定的，玩久熟練後，用最高時速從頭飆到尾也不是不可能的事。雖然以現今
      眼光來看，其畫面聲光上都已落伍，但其實還是蠻好玩的一個遊戲。
 
備註：本版為1.1版

操作：

1.選車：上下鍵選擇，Enter開始遊戲，Esc結束遊戲
2.遊戲：
  左右鍵 = 轉彎
  上鍵 = 加速（加油）
  下鍵 = 減速（剎車）
  9 = 右轉並加速
  7 = 左轉並加速
  3 = 右轉並減速
  1 = 左轉並減速
  A = 換高速檔
  Z = 換低速檔
  D = 排檔狀態顯示切換
  P = 暫停遊戲切換
  Ctrl-Q = 關掉音效
  Ctrl-S = 打開音效

說明：當出現警車時，保持110公里左右，讓警車超不過去，一路追到加油站，可以得到額外
      的分數。
