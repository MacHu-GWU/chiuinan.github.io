名稱：惡魔城1 (Castlevania，英文版)
簡介：Konami 1986年出品的動作遊戲，1990年移植到DOS，2002年再移植到Windows，收錄在
      「惡魔城與魂斗羅系列合集（Konami Collector's Series: Castlevania & Contra）」
      [待補]。
 
保護：DOS版為手冊密碼，破解碼：
      CCGA.EXE 或 CEGA.EXE
        7E 04 FF 1E
        -- -- EB 02
備註：DOS版電腦速度不可太快，否則進入遊戲可能當機跳出。
檔案：CC.rar = 惡魔城與魂斗羅系列合集（已免光碟）
操作：

一、DOS版

Space = 攻擊
上鍵+Space = 使用特殊武器攻擊
Enter = 跳躍
Ctrl+S = 存檔
Ctrl+Q = 離開

二、Windows版

F = 開始
A = 攻擊
上鍵+A = 使用特殊武器攻擊
S = 跳躍
F1 = 系統選單
