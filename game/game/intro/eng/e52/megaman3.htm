名稱：洛克人3：機器人起義 (Mega Man 3: The Robots Are Revolting，英文版)
簡介：Hi-Tech 1992年出品的動作遊戲，由CAPCOM前員工仿造紅白機版洛克人所開發設計。
      遊戲中共有6個關卡可選擇 [待補]。
 
保護：無
備註：電腦速度不可太快，否則一下子就死了。
備註：本系列並沒有二代，因為本遊戲的機器人大師和CAPCOM紅白機版洛克人3的火花人相似，
      故直接取名為三代。
密技：在遊戲設定畫面中按N放開後再按下D，然後才Start Game，則進入遊戲成為不死之身。
操作：（若要使用鍵盤，務必在遊戲設定畫面，將Joystick設成OFF）

Space = 射擊（Fire）
J = 跳躍
Esc = 切換武器
F9 = 暫停遊戲
F10 = 結束遊戲

說明：可選的關卡有：

1.Torch Man
2.Bit Man
3.Blade Man
4.Shark Man
5.Oil Man
6.Wave Man
