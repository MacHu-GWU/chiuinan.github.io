名稱：洛克人1 (Mega Man，英文版)
簡介：Hi-Tech 1990年出品的動作遊戲，由CAPCOM前員工仿造紅白機版洛克人所開發設計，
      但獲得眾多負面評價，更被評為有史以來最糟糕的電腦遊戲之一。遊戲中共有3個關卡
      可選擇 [待補]。
 
保護：無
備註：電腦速度不可太快，否則一下子就死了。
密技：在遊戲設定畫面中按N放開後再按下D，然後才Start Game，則進入遊戲成為不死之身。
操作：（若要使用鍵盤，務必在遊戲設定畫面，將Joystick設成OFF）

Space = 射擊（Fire）
J = 跳躍
Esc = 切換武器
F9 = 暫停遊戲
F10 = 結束遊戲

說明：可選的關卡有：

1.Sonic Man
2.Volt Man
3.Dyna Man
