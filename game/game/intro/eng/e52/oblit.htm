名稱：毀滅戰士 (Obliterator，英文版)
簡介：Psygnosis 1989年出品的動入遊戲，台灣由智冠軟體世界代理發行 [待補]。
 
保護：無
操作：

左右鍵 = 左右走
上鍵 = 坐電梯向上／進入傳送門
下鍵 = 坐電梯向下
Space = 向下發射武器
Enter = 切換第一控制圖形或第二控制圖形

1.第一控制圖形

F1 = 使用手槍（Pistol），右邊數字為剩餘彈藥
F2 = 使用來福槍（Rifle），右邊數字為剩餘彈藥
F3 = 使用爆破槍（Blaster），右邊數字為剩餘彈藥
F4 = 使用火箭砲（Bazooka），右邊數字為剩餘彈藥
F5 = 引擎（Engine）
F6 = 防禦系統（Shields）
F7 = 武器系統（Weapons）
F8 = 梭（Shuttle）
F9 = 資料帶（DataPack）

2.第二控制圖形

F1 = 向左走
F2 = 坐電梯向上／進入傳送門
F3 = 坐電梯向下
F4 = 向右走
F5 = 停止移動
F6 = 進入防禦罩之門，可增加防護罩能量
F7 = 向前跳躍
F8 = 停止行動並射擊（由方向鍵控制射擊方向）
F9 = 防禦／翻滾（配合左右方向鍵）
F10 = 拾起物品／清除物品／插入物品
