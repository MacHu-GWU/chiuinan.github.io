名稱：幻想新國度 (Asterix and Obelix XXL，英文版)
簡介：Atari 2004年出品的動作冒險遊戲，改編自法國知名漫畫「Asterix」 [待補]。



保護：光碟
備註：移機後，需執行光碟ELBdrivers目錄裡的ELBdrivers.exe，安裝所需驅動程式。
檔案：nocd.rar = 免光碟檔
操作：

上下左右鍵 = 移動
F = 動作
Space = 跳躍
P = 暫停（系統選單）
G = 攻擊敵人
D = 使用特技
R = 換Obelix
C = 換Dogmatix
I = 顯示狀態
Enter = 確認
Esc = 取消
右Shift = 鏡頭往上
左Shift = 鏡頭往下
Ctrl = 旋轉鏡頭
V = 鏡頭對往角色方向
