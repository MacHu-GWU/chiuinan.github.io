名稱：機甲戰神 (Interrupt，英文版)
簡介：Family Production 1995年製作的動作戰鬥遊戲，由歡樂盒代理發行 [待補]。
 
保護：數字密碼，破解碼：
      INTER.EXE
        C7 06 B2 04 09  (不出現密碼)
        -- -- -- -- 00
      或
      TITLE.EXE
        0B C0 75 09  (密碼隨便輸入)
        31 -- -- --
操作：

Alt = 攻擊
Ctrl = 跳躍／飛行
Enter = 查看等級
F1 = 暫停
F2 = 音樂開關
F10 = 結束離開
