﻿名稱：驚爆四川麻將 (中文版)
簡介：新造科技1999年出品的18禁四川省麻將遊戲，玩者必須在限時內拿掉所有可兩折（三
      線段）內連接的相同牌組。遊戲中也提供了一些輔助過關的功能。當整個牌組解開後，
      便會顯示一張美女圖做為獎賞。遊戲一共有30個關卡，每過過關時，剩餘的時間做為
      下一關使用。整個遊戲基本上有一定的娛樂性，算是可玩的一款遊戲。



保護：光碟，直接在光碟上執行。
修改：SICHUAN.EXE（可拷貝出來單獨執行，會自動讀取光碟上資料）

1.寶石不減
  FE C8 5F A2 50 C9 41 00
  90 90 -- -- -- -- -- --
2.時間到不結束
  76 09 C6 05 09 BF 41 00 14
  EB -- -- -- -- -- -- -- --

資訊：美女圖檔NEW4R\BG\TENG

各byte資料讀取後，需向左滾轉（rol）3個位元才是實際資料。各影像資料為：

檔頭資料：18 byte
  不明：12 byte
  影像水平大小：2byte，280h=640
  影像垂直大小：2byte，1E0h=480
  不明：2 byte
影像資料：每個點3byte，BGR，由最後一行開始往上放（640x480x3=921600=E1000h）
檔尾資料：不明，209h byte

備註：CG圖無露點。
操作：

滑鼠左鍵 = 確定選牌
滑鼠右鍵 = 取消選牌
F5 = 使用一顆寶石，重排牌張

說明：

1.共有10位美女，每個美女3關，共30關。各美女的難易度不同。
2.過關即可顯示一張美女圖，剩餘的時間繼續做為下一關的限時。
