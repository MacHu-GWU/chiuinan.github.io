﻿名稱：Taiwan麻雀／台灣麻將 (中文版)
簡介：向量科技1999年出品的16張麻將遊戲。遊戲提供了16位人物供玩家進行自由對戰，並
      未提供任何輔助的道具，為一單自由對戰的麻將遊戲。電腦的牌技不差，要獲勝需要
      運氣。幸好遊戲可隨時存檔，玩家可藉由存取檔來調整打法，以取得優勢，娛樂性還
      算可以。



保護：光碟，破解碼：
      Inetmj.exe
        83 F8 05 74 2B
        -- -- -- EB --
        83 BD D8 FE FF FF 05 0F 85 87 00 00 00
        -- -- -- -- -- -- -- 90 90 90 90 90 90
      必須放在\INETMJ目錄裡執行，並拷貝光碟裡的Index.Dat。可使用下列方式模擬硬碟：
        subst x: .
      結束後再以下列方式取消：
        subst x: /d
修改：存檔INetMjS?.CKC

位置0860h = 各家籌碼（我方第一個），均4 byte
位置0934h開始為手中的牌，每張4 byte，共16張。各牌的代號為：
  11(0Bh)～19(13h) = 1～9萬
  21(15h)～29(1Dh) = 1～9索
  31(1Fh)～39(27h) = 1～9筒
  41(29h)～44(2Ch) = 東南西北
  51(33h)～53(35h) = 發中白

操作：滑鼠右鍵 = 吃碰槓胡選單/取消代打
說明：

1.個人競技賽，玩到籌碼不足的人即退出，換人上場並重新開局。一直玩到牌腳不足為止。
2.打牌若面臨抉擇時，可存檔後再打，若覺抉擇的結果不太好時，可讀檔回來換另一種打法
  試試看。
