名稱：城堡幻想曲2外傳：阿里哈特戰記 (簡中版)
簡介：Studio e.go 2005年出品的遊戲，主要目的是補充城堡幻想曲2聖魔大戰之前的劇情，
      因此遊戲中並沒有任何戰鬥，只是單純的劇情故事書 [待補]。



保護：無，可直接在光碟上執行
操作：

滑鼠左鍵/Enter = 顯示下個對話
滑鼠右鍵/Space = 暫時不顯示對話框
上下鍵 = 顯示之前的對話
Ctrl = 快速略過劇情
F1 = 關於本遊戲
F3 = 快進模式開關
F7 = 系統信息
F8 = 聲音設定
F9 = 信息設定
F10 = 其他設定
F11 = 聲音設定
F12 = 自動模式開關
Alt-Enter = 全螢幕切換
Alt-F4 = 結束遊戲
存取檔只能在視窗模式的選單中選取
