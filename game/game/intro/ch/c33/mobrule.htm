名稱：黑金企業 (Mob Rule，中文版)
簡介：System 3 1999年出品的即時經營模擬策略遊戲，由會宇中文化並代理發行 [待補]。



保護：光碟，破解碼：
      MR.exe
        B8 80 65 4B 00
        EB 19 90 90 90
備註：非安裝者，必須修改SETTINGS\INSTALL.DAT裡的相關路徑。其中Language值如下：

0 = 英文
6 = 外部定義語言（如中文等）

備註：如果在主畫面選單時，滑鼠游標移動卡頓，請將MR.exe設成Win9x相容。
資訊：DATA.FIL格式

分檔資訊數目：4 byte（與3BD7A59Ah互斥），包含最後一個終止資訊
分檔資訊：每個17 byte（先經過資料解密）
  檔名：13 byte（含NULL字元）
  開始位置：4 byte（結束位置由下一個分檔資訊的開始位置得知）
分檔資料：無加密

*** 資料解密法

int index = 0
while 還有資料
{
  a = 讀取1 byte
  b = ((((a-0x27)  & 0xFF) ^ 0xA5) - 0x1B - index) & 0xFF
  index = index + 1
  輸出b
}
