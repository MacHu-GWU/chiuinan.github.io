名稱：皇帝 (中文版)
簡介：全崴1996年出品的皇帝模擬遊戲，遊戲的主要目標在於玩家扮演皇帝的一生，而於皇
      帝死後對其進行評價。遊戲過程包括政務、軍事、後宮等等各種皇帝所常遭遇的事物。
      遊戲本身則很像一般的策略遊戲，只是其目標並非消滅各國、統一天下，而是必須玩
      完皇帝的一生。剛開始玩時或許會感到新鮮，但玩到後來會因為其一成不變的過程，
      而覺得厭煩。應該說是這個遊戲的突發、隱藏與隨機事物太少了些。不過若單純以當
      皇帝過過癮的眼光來看，這個遊戲還算是蠻成功的。

攻略：（作者－青衫）

遊戲的目的在於當皇帝的過程，因此切莫窮兵贖武去消滅他國，以治理本國為優先（不過那
也是一種結局）。以下依主畫面中各地點，說明各選項的注意地方。

1.宣政殿（早）

  (1) 每次進來都聽聽宰相的話，他會告訴你應注意的地方。
  (2) 招募士兵後，還要進行武器購買。沒有武器裝備，士兵是無法派出場的。至於是要派
      騎兵、槍兵或是步兵攻防，其實效果並沒有太大差異。武器裝備的多寡和士兵的數目
      都會影響軍力。
  (3) 官員人才很重要，因此要常舉行制舉招募人才。不過在初期金錢不太夠，先不要招太
      多人才，夠用就好，而且必須魅力超過250以後，才能招到名臣。屬性不佳的官員，可
      用官員外調去除。不過如果想去消滅他國的話，便必須準備兩種閒官，一是進攻對方
      的武將（武力要高），一是佔領後做為該地刺史和將軍的人才，因此不要將一些還算
      不錯的人才都去除了，否則將來便無法再招募到。通常忠誠度小於60（封爵到王時可
      加忠誠度30），或是文官清廉度小於80的人，都可以將他去除掉。另外，官員在55歲
      以後即退休，因此也要準備一些散官隨時接替。
  (4) 官職調動時，儘量將優秀的人才調往地方，地方官職的忠誠度最好都大於90以上：
      a.刺史：清廉、智力高的去當，智力低會影響開發，清廉低會影響稅收。刺史的忠誠
              度也要高。
      b.將軍：忠誠最重要，否則可能叛變，武力不是最重要，只影響士兵訓練度而已，但
              最後都能訓練至99。至於清廉度則沒什麼影響。
      c.中央將軍：找最高武力的來當，訓練會比較快。
      d.吏部尚書：找經驗值最高的來當，會影響每次制舉的招考人數。
      e.兵部尚書：找經驗值高的來當。
      f.其他官員：隨便阿狗阿貓都可以，沒什麼影響（包括忠誠度與清廉度）。
      如果各官員的忠誠度和清廉度都高的話，便不必給刑部尚書調查經費了。至於地方官
      儘量委任，避免耗用太多體力，但記得每半年調回中央直轄一次（宰相會提醒）。
  (5) 特赦雖可增加皇威，但會降低治安值，不值得。至於祭祀雖可降低災害發生機率，但
      無法全部消除，災害仍會隨機發生，也有些不太值得。另外，偶爾要派中央將軍進行
      安全檢視，加強京城安全工作，否則晚上睡覺時，很有可能會遇刺身亡。
  (6) 領地的士兵數目要大於1/4的人民數，否則可能盜賊叢生，影響治安。查閱軍事總表，
      出現紅色者即為士兵數不足。
  (7) 煉丹用的物品必須靠他國進貢，因此儘可能和其他國打好外交關係。可先簽下互不侵
      犯條約，然後是防禦同盟，最後是保護，尤其是東突厥，用多少錢都要把他安撫，不
      過要簽約，特別是保護，自己本身的軍力要夠高，否則沒人鳥你。同時別忘了跟他們
      貿易，雖然一開始會虧錢，不過到最後，幾乎不用向自己人民徵稅就有用不完的錢了。
      一開始可先少量貿易，等轉虧為盈時，再調到最高，不過額度和外交關係有關。如果
      有女兒滿16歲時，不妨拿她來和親，反正放著也沒用。當外交關係達到99時，每次進
      入宣政殿便會隨機得到進貢物品。如果與所有國家都簽下保護，便會被尊稱為天可汗
      皇帝。
  (8) 每次選項動作都會損耗一些體力，要特別注意不要亂選，體力用盡可是會昏倒的。最
      好儘快利用狩獵增加最大體力值。
  (9) 進入宣政殿，每次要消耗10點體力、5點健康、5點快樂，如果確認沒什麼政務的話，
      也可不進入。

2.金龍殿（早、午、晚）：恢復體力150、健康5。
  後宮（早、午、晚）：恢復體力（數量視嬪妃能力而定）、快樂5

  後宮雖然可加快樂，提升體力的數目也比較多，但折損壽命的速度也比較快，因此沒事不
  要儘往後宮跑，免得結局變成荒淫好色的皇帝。後宮嬪妃若懷孕時，便不能再做了，可進
  行宮女安排，找新的嬪妃。通常知名的嬪妃，能力都比較強（即恢復體力的值較多），太
  差的不必將之冊立。另外，嬪妃的能力會隨著她的經驗而提升，因此儘量精而不要廣。另
  外，生子、封禪等事件以及出巡等，均可恢復健康與快樂，不一定要在後宮勤做。

3.紫宸殿（午）

  (1) 若之前有冊立嬪妃，可在此處調整，通常將嬪妃的職位往上升，其能力也會隨之加強。
      另外，若長時期處理政務，以致下午都沒空時，懷孕的嬪妃將無法產子，若同時有多
      個嬪妃懷孕待產，當皇帝下午的時間空出來時，則只會產出一子（很怪的設計？！）。
      此時只要進出紫宸殿幾次，便能將所有子女生下來，而嬪妃才能重回臨幸行列中。
  (2) 遊戲過程中也有記錄和子女的互動關係，因此沒事時也要偶爾來召見子女看一看。但
      每次進入紫宸殿會耗損10點體力值，因此要探視可集中召見數次，遊戲中只記錄召見
      的次數而已。
  (3) 狩獵是唯一可增加最大體力值的地方，要優先提升。至於季節影響並不大，主要看射
      中的獵物數目。體力提升至最大值500，即有封號。

4.武道場（早、午）：增加武術
  御書園（早、午）：增加智慧
  棋琴閣（早、午）：增加魅力
  百劇院（早、午）：增加才藝

  屬性的多寡會影響到結局的評價，同時也與政務有所影響。魅力會影響人才招募。各屬性
  最大值為500。

5.慈寧宮（午）

  可覲見太后，會影響結局對尊長的關係。由於皇太后在中期便會逝世，因此初期便要常來
  覲見。

6.青爐房（午）

  唯一可增加壽命的地方，煉丹物品必須經他國進貢。至於煉丹方式如下：

  A.冰山雪蓮
  B.冷凝不化冰
  C.千年人參王
  D.透光靈芝
  E.神僧舍利子
  F.龍腦香
  G.鯨王脂
  H.熾火蛙

  A+B = 體力現值+100
  C+D = 體力現值+100
  B+D+F = 健康+10、快樂+10
  B+D+E+H = 武術、智慧、魅力、才藝各加兩點
  A+B+D+E+H = 最大體力值+30
  A+B+C+D+E+F = 壽命+約20
  C+D+E+F+G+H = 壽命+約20
  A+B+C+D+E+F+H = 壽命+約30
  任意七種 = 壽命+約25
  全部八種 = 壽命+約45
