名稱：EQ大富翁 (中文版)
簡介：弘煜1998年出品的大富翁遊戲，由智冠代理發行。遊戲結合了問答益智遊戲 [待補]。



備註：本版為1.20版
保護：無，需放入光碟才有背景音樂
備註：安裝時，硬碟空間不可過大，否則可能會被誤判空間不足，無法安裝。或是進行下列
      修改：
      SETUP.EXE
        7D 2A 68 B4
        EB -- -- --
        7D 55 68 B0
        EB -- -- --
備註：如果遊戲無法正常執行，可使用DxWnd執行（D3DWindower不行）。
檔案：quiz_patch12.rar = 1.20更新檔
密技：

1.轉輪盤時，依次按下F11、F3、F3、Enter，再按下列按鍵：

F1 = 抓圖
F5 = 查看大地圖（可捲動），再按一次解除
F9 = 所有問題的答案自動出現

2.轉輪盤時，依次按下F11、F6、F7、F5、F8、Enter，再按下列按键：

F3 = 玩打盤子的遊戲
F4 = 玩抽牌的遊戲
F6 = 玩踢足球的遊戲
F7 = 玩爆雷飛行的遊戲
F8 = 直接過關

操作：進入遊戲後，將滑鼠移到頂端，即可出現系統選單。
說明：

1.選人物時，要點選直到出現1P或2P，才能按下START開始遊戲，否則會全部都是電腦操控。
