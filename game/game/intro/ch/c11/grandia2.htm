名稱：冒險奇譚2 (Grandia 2，中文版)
簡介：Game Arts 2000年出品的角色扮演遊戲，於2002年移植到PC，中文版由松崗代理發行，
      以3D畫面呈現，採團隊回合方式戰鬥 [待補]。



備註：本版為2002年繁中光碟版（韓語配音）。
保護：光碟
備註：需Direct3D 7，可使用VMware+XP執行。
檔案：GRANDIA.II.HD.Remaster.v26.05.2022.rar = Steam繁中1.02版
操作：可用GmConfig.exe變更設定：

上下左右鍵 = 移動
Enter = 走路/跑步切換
Home/PageUp = 旋轉視角
Z = 確定/動作
X = 取消
Tab = 羅盤開關
Esc = 系統選單
F12 = 離開遊戲

說明：存檔必須找到圓錐形光圈，走進光圈按確定鍵(Z)便會能存檔+恢復。
