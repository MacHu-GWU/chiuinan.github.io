名稱：伊卡斯特傳說 (Legend of the Icaston，中文版)
簡介：宏申1993年出品的角色扮演遊戲，由第三波代理發行，以3D斜角畫面呈現，團隊回合
      制戰鬥 [待補]。
 
保護：顏色密碼，破解碼：
      3DRPG2.EXE
        8E D8 E8 60 4C  (略過密碼輸入)
        -- -- 90 90 90
        75 36 8D 16 C4 00
        90 90 -- -- -- --
      或
        75 CB 73 C9  (直接輸入正確密碼)
        90 90 90 90
        75 36 8D 16 C4 00
        90 90 -- -- -- --
      或
        E8 7D 00 E8 7A 00  (直接輸入正確密碼)
        EB 0E 90 -- -- --
操作：

Space = 確定／系統選單／略過開頭動畫
Ctrl = 取消／戰鬥選單

說明：

1.必須在原野才能存檔。
2.遇敵時，若沒叫出戰鬥選單，則時間一到便會自動進行一般攻擊。
