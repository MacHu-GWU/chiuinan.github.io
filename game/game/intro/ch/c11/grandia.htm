名稱：冒險奇譚 (Grandia，中文版)
簡介：Game Arts 1997年出品的角色扮演遊戲，於2019年移植到PC，到2022年才有繁體中文
      版。以3D畫面呈現，採團隊回合方式戰鬥 [待補]。



備註：本版為Steam繁中1.02版，建議設定語音為日文（也可在遊戲中更換）。
操作：可用Launcher.exe變更設定：

上下左右鍵 = 移動
Q/E = 旋轉視角
1/3 = 移動羅盤
W = 確定/動作(如交談、撿取等)
A = 取消/結束/跑步(按著)
S = 暫時拉近視角觀看
D = 裝備選單
Tab = 地圖
Enter = 開始/略過開頭動畫/設定選單

說明：存檔必須找到圓錐形光圈，走進光圈按確定鍵(W)便會能存檔+回復。
