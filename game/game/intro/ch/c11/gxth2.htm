操作：（遊戲不支援滑鼠，請使用鍵盤操控）

右Ctrl+左右鍵 = 平移
X/Enter = 確認／開門
Z/. = 取消／紮營指令（隊伍選單）
Esc = 略過開頭劇情
A/* = 切換角色視窗
S/- = 開啟特殊指令
F1 = 快速存檔
F5 = 按鍵說明
F6 = 開關血液一覽
左Ctrl = 使用咒文
右Ctrl = 移動時固定面對方向／切換角色
左Shift = 使用道具／切換地圖樓層
右Shift = 自動顯示地圖（需有地圖道具）／切換地圖樓層
