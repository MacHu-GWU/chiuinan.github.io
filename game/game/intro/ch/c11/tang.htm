名稱：大唐三藏 (中文版)
簡介：廈門三馳工作室2000年開發的角色扮演遊戲，由金智塔代理發行，以3D斜角畫面呈現，
      團隊回合制方式戰鬥 [待補]。



保護：無
備註：遊戲執行後，若畫面顏色亂掉或閃爍，請使用DDrawCompat、dgVoodoo2或D3DWindower。
密技：

1.在物品界面中使用木魚，敲30次後，可以補滿人物的體力。
2.在物品界面中重複看血書，可以補充人物的靈力，直到補滿。
3.在物品界面中閱讀多心經，可以補充人物的內在力量。
4.遊戲中放開鼠標（別碰到它），然後按F1鍵，畫面左下角會出現一個冒號，輸入以下文字：

POWERHP #1 #2 補充主角#1體力#2點（主角編號由0開始）
POWERMP #1 #2 補充主角#1法力#2點
POWEREP #1 #2 補充主角#1靈力#2點

操作：Esc = 遊戲選單
