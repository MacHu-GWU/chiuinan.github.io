資訊：修改Windows下的Corum3.ini：

DevMode=是否進入偵錯模式，內定為0

操作：

上下左右鍵 = 移動（雙按=跑步）
Z = 防禦／取消
X = 攻擊／確定
C = 跳躍(凱恩)／滑步(艾舒莉亞)／舉起物品(西比)
V或Esc = 功能選單
1～5 = 使用物品（需先設定對應的物品）
A = 顯示人物狀態
D = 顯示人物狀態及物品
S = 切換人物
F1 = 切換人物為凱恩
F2 = 切換人物為艾舒莉亞
F3 = 切換人物為西比

說明：必須在特定地點（金色旋轉體）上按X鍵才能存檔。
