名稱：時空魔域 (Wander Wonder，中文版)
簡介：COMPILE 2000年出品的角色扮演遊戲，由TGL中文化並代理發行，以2D畫面呈現，採單
      人即時方式戰鬥 [待補]。



保護：無
備註：遊戲另附Wander Number（WanderN目錄）。如果無法正常啟動，請使用DxWnd執行。
操作：

F5 = 回到標題畫面
上下左右鍵 = 移動（連按兩次為加速）
Space = 確定／開啟選單
Esc = 開啟選單
Z = 攻擊／動作／結束交談（新遊戲顯示「冒險的序章」標題後，要按下才會繼續遊戲）
X = 跳躍
C = 變更裝備，上下鍵換形態，左右鍵換裝備
A或V = 防禦
Ctrl = 加速

存檔要到旅館休息後才能存檔。

說明：

*** Wander Number玩法

一、畫面說明

1.場地（數字）：下方（紅色）是我方場地，上方（藍色）是對手的場地。
2.家（桶）：我方在右邊，對手在左邊，裡面的數字便是此回合分數。
3.得分（板）：右下是我方總分，左上是對手總分。

二、規則說明

1.散播：選擇我方場地上的數字，該數字會變成0，然後以逆時針方向，將後面該數字的位置
  （顯示為虛藍色），每個數字均加1。亦即將選擇的數字往後逐一散播出去，直到用完為止。
2.重播（Replay）：如果我方家下方出現此字樣時，表示散播完後可以再玩一次。
3.獎金（Bobus）：如果有桶子下方出現此字樣時，則散播後，該位置上下桶子的數字均會加
  到我方家的分數。
4.結束：當我方場地或對手場地的所有數字均為0時，該回合結束。
5.勝負：全部回合比完後，依照雙方總分決定勝負。
