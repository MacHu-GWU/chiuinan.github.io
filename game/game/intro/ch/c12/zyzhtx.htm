名稱：三國趙雲傳之縱橫天下 (中文版)
簡介：第三波2004年出品的角色扮演遊戲，採斜角3D畫面呈現，以單人即時方式戰鬥。本遊戲即為三國趙雲傳的續集 [待補]。



保護：無，若遇光碟無法存取的訊息，取消即可。
備註：必須特定顯示模式才能執行，使用DDCompact或D3DWindower均無效，可先在實體機上
      執行看看，只要能看到開頭動畫便可以了。開頭動畫後若黑屏，試著切至其他視窗或
      桌面再切回遊戲看看。若仍無法顯示，便修改Game.ini內容，設定ScreenMode=0，並
      將Game.exe設成16位元減少色彩模式，再重新執行執行即可（視窗模式）。
備註：非安裝遊戲進行移機時，要進行下列registry註冊，否則遊戲會陷入無限循環。

[HKEY_LOCAL_MACHINE\SOFTWARE\Acerplay\三國趙雲傳之縱橫天下]
"AppPath"="TARGETDIR"
"CDDrive"="SRCDISK"

[HKEY_LOCAL_MACHINE\SOFTWARE\Acerplay\三國趙雲傳之縱橫天下\1.00.000]

操作：

滑鼠左鍵 = 行走/對話/拾取物品/普通攻擊
滑鼠右鍵 = 必殺技攻擊
方向鍵 = 行走
Shift+滑鼠左鍵/方向鍵 = 跑步
Ctrl+滑鼠左鍵 = 原地攻擊
Sapce = 對話/拾取物品
A = 裝備弓箭
D = 裝備長槍
S = 裝備長劍
1～8 = 使用物品
Esc = 系統選單
Tab = 小地圖開關
