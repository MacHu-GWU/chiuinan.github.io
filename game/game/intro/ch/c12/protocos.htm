簡介：韓國YoYo Games 1999年出品的角色扮演遊戲，由會宇中文化並代理發行，以2D捲軸畫
      面呈現，採單人即時方式戰鬥 [待補]。



保護：光碟，破解碼：
      Protocos.exe
        83 F8 05 75 65
        -- -- -- EB 6E
備註：遊戲需要320x240x16顯示模式，現今大部份的顯示卡都已無法支援。若直接執行出現
      錯誤或顯示不正常，請使用DxWnd、D3DWindower或dgVoodoo等工具協助執行。
修改：Protocos.exe

  -- -- -- -- -- -- -- -- FF -- --

  EB -- -- -- --

  75 12 DD 05 88 (共5個)
  EB -- -- -- --
5.Alt鍵按著可以連續攻擊
  0F 85 F0 07 00 00 83 3D 98 12 99 00 00 0F 85 E3 07 00 00
  90 90 90 90 90 90 -- -- -- -- -- -- -- 90 90 90 90 90 90

操作：

左右鍵 = 移動
Alt = 攻擊
Ctrl = 跳躍
下鍵 = 防禦 (MP減少0.02)
上鍵 = 開門進屋／打開寶箱
Shift = 切換召喚獸（畫面右側Flare、Galaxy等即是）
Space = 交談
Enter = 功能選單
F1 = 狀態顯示開關
F2 = 背景圖顯示開關
F3 = 垂直同步開關，電腦速度慢者可增加速度
F10 = 結束遊戲
