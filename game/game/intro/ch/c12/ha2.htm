名稱：聖女之歌2：撒雷母天使 (Heroine Anthem 2: The Angel of Sarem，中文版)
簡介：唯晶科技2003年出品的角色扮演遊戲，由智冠代理發行，以3D斜角畫面呈現，採團隊
      即時方式戰鬥 [待補]。



檔案：本版含「天使樂章」
保護：光碟StarForce 3.03
備註：Ha1ToHa2.exe為一代存檔轉二代，SetDXMode.exe為設定執行配備
檔案：聖女2更新檔1(天使樂章).rar = 「天使樂章」資料檔
      聖女2更新檔2.rar = 「天使樂章」更新檔
      聖女之歌2破解檔.rar = 破解檔（需將第四片光碟檔案拷入安裝目錄，才能免光碟）
密技：進入選單畫面，也就是按滑鼠右鍵進入的畫面，按Enter鍵，輸入以下密技，輸入完後
      按Enter鍵即可，成功會有音效（驚嘆號請用左Shift+1按出）。

mana stone!	  符石全滿
windthunder!  進入戰鬥配置人物完成後，持續按下左Alt、左Shift、K三個鍵，可立即獲勝
heal me!      狀態回復

操作：

Tab = 開啟小地圖
Ctrl+R = 走路／跑步切換
滑鼠右鍵 = 開啟選單
Alt+1 = 魔法選單
Alt+2 = 裝備選單
Alt+3 = 狀態選單
Alt+4 = 系統選單
Alt+5 = 讀檔選單
Alt+6 = 存檔選單

戰鬥時，按滑鼠右鍵出現選單，選右上圖案決定要設定的人員，再設定其他要做的動作，如
上方圖案的攻擊等。最後再按滑鼠右鍵回到戰場，開始動作。
