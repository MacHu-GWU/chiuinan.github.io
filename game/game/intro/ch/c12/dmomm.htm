名稱：魔法門：黑暗彌賽亞 (Dark Messiah of Might and Magic，中文版)
簡介：Arkane 2006年出品的第一人稱3D單人動作角色扮演遊戲，由Ubisoft代理發行，台灣
      則由英特衛代理發行繁中版 [待補]。



保護：光碟SecuROM 7.30，可用SRLoader啟動（勾選繞過遊戲自校驗）
備註：VirtualBox+XP的顯示畫面會過暗，虛擬機建議使用VMware。
備註：Win64系統必須進行下列修正才能啟動遊戲（但不確定是否適用所有Win64系統）：

1.搭配dgVoodoo2，將其檔案拷入安裝目錄中。
2.利用批次檔或捷徑，以"mm.exe -novid"方式啟動遊戲（即不播放動畫）。

不過打開背包，滑鼠經過戰鬥技能圖案時，仍會當機，不是很穩定，不推薦。

備註：非安裝移機者必須進行下列Registry登錄，否則執行後會變英文版：

[HKEY_CURRENT_USER\Software\Valve\Steam]

"Language"="tchinese"

備註：mm.exe執行參數：

-novid  不顯示動畫

檔案：nocd.rar = 免光碟檔
操作：

W/S = 前進／後退
A/D = 左／右平移
Alt+W = 步行
Shift+W = 奔跑
Space = 跳躍
Ctrl = 蹲下
Z/X = 向左／右閃
滑鼠左鍵 = 攻擊
滑鼠右鍵 = 防禦
滑鼠滾輪 = 變更技能
F = 踢擊
V = 使用特殊能力
R = 放大視野
Tab = 檢視背包
E = 動作（撿起／開啟等）
1～9 = 使用腰帶物品
O = 顯示任務目標
F6 = 快速儲存
F9 = 快速載入
