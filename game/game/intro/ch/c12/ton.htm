名稱：江南才子唐伯虎 (中文版)
簡介：智冠1998年出品的即時制戰鬥角色扮演遊戲，並具有解謎與養成的成份。遊戲並沒有
      迷宮，玩家必須到各個固定地點與人文試武鬥，以提升自己等級與屬性，並使用物品
      以觸發劇情，最終目標便是追求到八位女主角 [待補]。



保護：光碟，破解碼（放入光碟才有背景音樂）：
      TIGER.EXE
        74 05 E8 03
        EB -- -- --
備註：在Windows系統上安裝時會檢驗C:\WINDOWS裡有無ton.ini，有就是有安裝過，無法再
      安裝。XP以上系統安裝前要設成Win9x相容，但安裝後仍是以DOS模式執行。其實安裝
      內容幾乎就是整個光碟的內容（除了極少數安裝專用的檔案外），因此若安裝有問題，
      其實可以直接將光碟內容拷到安裝目錄，再執行SETSOUND.BAT設定環境。
密技：先以下列命令啟動遊戲（會有3聲警笛聲）：

fix64v tiger.exe GODTCW

在戰鬥中可以按下：

F1 = 戰鬥時不損血 
F2 = 獎金變多 
F3 = 顯示座標 
F5 = 顯示場景名稱 
F6 = 顯現出場景中的寶物
F8 = 戰鬥中直接勝利（但累積經驗值暫時看不到，要實際打一場才會有作用）
F9 = 無限使用絕招（不消耗文采、內功等值）
F10 = 離開遊戲
