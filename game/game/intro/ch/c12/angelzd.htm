名稱：天使之都 (中文版)
簡介：玩酷小組2001年出品的角色扮演遊戲，由宏碁代理發行，以3D畫面呈現，採單人即時
      方式戰鬥 [待補]。



保護：光碟+序號（ATT-0948-812）
備註：VirtualBox或VMware均無法正確執行遊戲，請使用PCem、86Box+Win9x系統，或是直接
      在Win64系統上執行（若無法執行，可使用dgVoodoo2）。
備註：進入遊戲後，可能會有近20秒的時間黑屏（等待錄音裝備反應），請耐心等候，或進
      行下列修改略過：
      Angel.exe
        E8 7A E3 FD FF
        90 90 90 90 90
操作：

左右鍵 = 轉身
上下鍵 = 前進後退（第三人稱）／上下看（第一人稱）
Ctrl+方向鍵 = 跳躍／水中前進
Alt+方向鍵 = 小步移動
Tab = 第三人稱／第一人稱
Space = 交談／橫掃（連按2次=刺擊，連按3次=迴旋攻擊）
Enter = 選擇魔法（按左鍵選物品，按右鍵選裝備）
1 = 使用彈弓模式
2 = 使用飛珠弓模式
滑鼠 = 進入彈弓模式後，鼠標即準星，按下滑鼠左鍵即可攻擊
F1 = 按鍵說明
F2 = 冒險日誌
F3 = 物品說明開關
F4 = 狀態表
P = 暫停
Esc = 離開彈弓模式／系統選單
Shift = 啟動語音辨識功能（需有錄音裝備）
