名稱：聖域破壞神／聖域 (Sacred，中文／英文版)
簡介：Ascaron 2004年出品的角色扮演遊戲，由昌哲科技中文化並代理發行，以斜角3D畫面
      呈現，採單人即時方式戰鬥。2005年再推出「魔都魅影（Sacred Underworld）」資料
      片，但並未中文化 [待補]。



備註：本版為繁中版Sacred Plus（1.8.2.136版），不含魔都魅影資料片。
保護：繁中版為StarForce 3.4（序號=K4896-USQS2-35PXE-RVAGN-29QTX），英文版為VOB 5.8
      英文黃金版(Gold,含資料)無保護
備註：需要Direct3D 7，若用虛擬機只能使用VMware。建議使用破解版，以便可在實體機上
      執行。若使用Daemon掛載光碟時，不能有實體光碟機。記得修改虛擬機的「vmx」檔，
      加入下列一行，否則會跳出不是支援的作業系統訊息：

monitor_control.restrict_backdoor = "TRUE"

備註：全螢幕時不能切回桌面，否則切回去時會畫面全黑。若全螢幕無法執行，可用Config.exe
      改成視窗模式，或試著設成32位元彩色深度。

備註：Sacred Plus免光碟檔使用方法：

1.修改checkfile.lst，移除sacred.exe（中文版無此檔，不需處理）。
2.將sacred.exe更名為sacred.org。
3.將破解檔sacred.exe和Caf.dll拷貝到安裝目錄裡。
4.Win7以後系統要將sacred.exe設定XP相容，或使用系統管理員權限執行。

密技：先增加執行參數/CHEATS=1，進入後按下~鍵開啟命令視窗，即可輸入：

sys cheats 963 = 啟用密技(原始版本)
sys cheats = 啟用密技(1.8版，即Sacred Plus)
cheat lord = 無敵
cheat teleport = 傳送
cheat tp = 傳送
cheat suicide = 自殺
cheat decap = 主角斷頭(純搞笑)
sys cheats 369 = 停用密技
sys cheats 任意數字 = 停用密技(1.8版，即Sacred Plus)
sys fps = 顯示框架速度
exit = 結束命令視窗

但使用密技後的人物，導出時圖像會被裝扮成兔子，以表示使用過密技（但可用修改器修掉）。

操作：

一、主選單說明

1.[導入]可使用之前導出的人物，角色的等級、屬性與裝備都會保留。
2.困難度一共有五種：
    Bronze（青銅）
    Silver（白銀）
    Gold（黃金）
    Platinum（白金）
    Niobium（鈮金）
  會影響敵人的強度，以及掉落物品的品質。在多人遊戲中均有其進入的等級限制。

二、遊戲操作

滑鼠左鍵 = 移動/攻擊(按住可持續攻擊)/動作(開箱,交談,撿取等)
Shift+滑鼠左鍵 = 走路/下馬/扔下物品/賣出物品(不必確認)
滑鼠右鍵 = 戰技/施法/組合技/使用或裝備物品/直接購買物品(不必確認)
Ctrl+滑鼠左鍵/右鍵 = 原地攻擊
Space = 使用治癒藥水
Q = 使用超渡藥水
W = 使用導師藥水
E = 使用解毒藥水
R = 使用專注藥水
Alt = 顯示所有可見人士或物品的名稱
A = 撿取所有物品(只在單人遊戲有效)
I = 背包欄
F = 戰鬥技藝欄
C = 連續技欄
L = 日誌
O = 遊戲選項
S = 儲存選單
F8 = 快速讀取
F9 = 快速儲存
Tab = 顯示小地圖
M = 世界地圖
+/- = 縮放可視範圍
Esc = 系統選單
P = 暫停
H = 輔助說明文字
Ctrl-B = 截取畫面
1～5 = 武器按鈕（等級到一定程度後才會增加）
6～0 = 戰技/法術/組合技按鈕（等級到一定程度後才會增加）
N = 隊伍信息(只在多人遊戲有效)
Enter = 打開聊天窗口(只在多人遊戲有效)
PgUp/PgDn = 聊天信息翻頁(只在多人遊戲有效)

1.右上人物畫像周圍紅圈為剩餘生命，綠色用來指示升下一級還需要的經驗值。
2.下方羅盤用來指示下個主線任務的方向。若有支線任務，則會有小的指針指向，如果有多
  個支線任務，可調出日記，點選尚未完成的支線任務，即可出現該任務的方向指針。
3.羅盤旁為目前設定的武器與戰技或連續技，可從背包、戰鬥技藝或連續技欄拖過來取代。
4.人物狀態欄[技能]或[能力值]左邊的按鈕（綠色／紅色）按下可進行切換。
