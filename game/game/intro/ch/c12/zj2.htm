名稱：日劫2：烈日奇俠傳 (中文版)
簡介：數位玩具2002年出品的角色扮演遊戲，由智冠代理發行。遊戲採用3D斜角畫面呈現，
      以單人即時方式戰鬥 [待補]。



保護：光碟+序號（9EB3E799G5G732G8），破解碼（需放入光碟才有背景音樂）：
      zj2.exe
        83 7D E4 00 74 02 EB 38
        -- -- -- -- -- 00 -- --
修改：zj2.exe

  2B 91 B0 02 00 00  (共2個)
  90 90 90 90 90 90

  8B 48 50 83 E9 01
  -- -- -- -- -- 00
3.修練點數不減
  8B 42 54 83 E8 01
  -- -- -- -- -- 00

密技：遊戲中按下~鍵，輸入love digitoy，再按下Enter鍵，即可啟動密技。之後可按下~鍵，
      輸入下列密技字串（輸入完要按Enter鍵）：

superman = 無敵
levelup = 升一級
openmap = 跳到大地圖
allplace = 打開所有地圖

注意密技只在同一個地圖有效，進入另一個地圖便必須重新再輸入一次。

操作：

滑鼠左鍵 = 攻擊／交談／開箱／撿拾（按住敵人可自動鎖定目標攻擊）
滑鼠右鍵 = 跳躍／使用物品（對物品欄）
Ctrl+滑鼠左鍵 = 使用副武器（投擲類或法器）
Shift+滑鼠左鍵 = 施展武功／心法（慕劍麒拿劍、赫連千羽拿刀才能施展武功）
F1～F4 = 使用物品
1～5 = 選擇武功／心法
Space = 切換武功／心法
Esc = 遊戲選單
Enter = 系統選單
Alt = 顯示地上物品
R = 跳步／走路切換
P = 位置資訊顯示切換

說明：

1.每個村落只會有5個NPC可交談。有時會遇到另一位主角，可以交談。
2.交談選項請慎選，會影響NPC的好感度，以及特殊事件是否開啟。
3.武功最後一招必須有特定事件後才能學習。
4.頭上出現的不良狀態符號：
  (1)骷髏：中毒，體力會持續減少。
  (2)盾牌融化：腐蝕，裝備耐久度會持續減少。
  (3)蛛網：定身，暫時無法移動。
  (4)符咒：封印，暫時無法施展武功心法。
  (5)剪斷懸絲：混亂，攻擊力暫停降低。
  可按F3使用丹藥（有兩種）或進入丹藥欄點選使用對應丹藥以解除不良狀態。
5.生剋原理為：水剋火、火剋風、風剋土、土剋水。
