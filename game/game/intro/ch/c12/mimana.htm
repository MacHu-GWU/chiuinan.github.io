名稱：魔導戰記 (Mimana Iyar Chronicle，中文版)
簡介：工畫堂2004年出品的角色扮演遊戲，2009年由娛樂通中文化，台灣由英特衛代理發行，
      以3D斜角視角畫面呈現，採團隊即時方式戰鬥 [待補]。



保護：光碟SecuROM 7.40，可使用SRLoader勾選繞過遊戲自校驗啟動。
操作：

1.平時

Ctrl = 快速略過劇情
F1 = 儲存檔案
F2 = 讀取檔案
Space = 交談／動作
S = 隊員交談
A = 系統選單
PgUp/PgDn = 上一頁（欄）／下一頁（欄）
Z = 確定
X = 取消

2.戰鬥

Space = 武器攻擊
F1～F4 = 魔法攻擊（由戰鬥設置中設定對應的魔法）
