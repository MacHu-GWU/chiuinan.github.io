名稱：神戒 (Hard To Be A God，中文/英文版)
簡介：英特衛2007年出品的角色扮演遊戲，採3D畫面呈現，以單人即時方式戰鬥 [待補]。



保護：無
備註：需要D3D加速，若無法正常執行，可修改<安裝目錄>\Profiles\<設定檔名稱>\local.ini，
      將r_fullscreen的值改成0。由於需要大量3D繪製，建議是在實體機裡執行，或是修改
      local.ini提升效能（參考密技部份）。
密技：先在<安裝目錄>\Profiles\<設定檔名稱>\local.ini的文字檔最後加上一行：

g_console 1

進入遊戲後，按"~"可開關控制台，輸入指令後再按Enter確認即可。可按Tab查看所有的指令，
以下指令，#=1表開啟，0表關閉：

god #                      無敵
r_fog #                    霧效果開關
r_grass_show #             草地繪製開關(關閉可以有效提升效能)
r_shadow_volume #          陰影渲染開關
g_collision #              穿牆模式[或使用no_clip #或noclip #]
g_megahealth #             復活已死的人物/動物[可對活物強制死亡]
g_skip_video #             跳過影片

另外遊戲的視角也可以修改，#=數值：

camera_distance_max #      設定最大觀看距離 [預設17]
camera_distance_min #      設定最小觀看距離 [預設7]
camera_angle_max #         設定對大觀看角度 [預設40]
camera_angle_min #         設定最小觀看角度 [預設30]
g_camera_fov #             改變視野         [預設45]

以上指令也可直接加進local.ini文件中，進入遊戲便立即使用。設定快捷鍵的方式：

bind DIK_ 

KEY是快捷鍵，可以設定A~Z、1~9、0、F1~F12、九宮格數字，ConsoleCommand則是指令。
例如bind DIK_K t_remove_grass，就是進入遊戲後按"K"即可移除草地。

操作：以下為內定按鍵，可經由選項修改：

WS = 前進/後退 (+左Shift = 跑步/走路)
AD = 向左/向右 (+左Shift = 跑步/走路)
R = 跑步/走路切換
Space = 跳起/防禦
Tab = 啟用/收起武器
移動滑鼠 = 旋轉視角
滑鼠左鍵 = 動作(如基本攻擊、交談、開箱、搜索等，對象目標亮起時)
E = 動作(含騎馬)
滑鼠右鍵 = 移動物品/強力攻擊
滑鼠滾輪 = 撞倒敵人
I = 物品欄
C = 技能點
L = 日記
Z = 鑽
M = 地圖
F5 = 快速存檔
F6 = 快速取檔
