前言

魔法門678整合版（Might and Magic 6,7,8 merge）係俄國Rodril以魔法門8為基礎，加入6代與7代的內容。英文版載點：

https://www.celestialheavens.com/forum/10/16657#p369458

必須先有原始的八代英文版，然後再加入上述整合版內容。中文語言包載點：

https://github.com/might-and-magic/mm678-i18n/releases/

由於中文語言包的版本比英文版舊了很多，因此下列3個Script檔不要覆蓋，比較不會出問題：

ExtraArtifacts.lua
MonsterPathfinding.lua
Quest_CrossContinents.lua

最後請再使用魔法門678三代整合版青衫繁中更新檔，以修正一些原版已變更的資訊，並採用原繁中版名詞。不過由於各代的繁簡中文版名詞本身並不統一，因此所有名詞將依次以六代簡中、八代繁中、七代繁中的次序，儘可能符合原版，或修改成簡單易懂者為準。

第一次執行時，務必執行MM8Setup.eng.Exe（英文版）選擇硬體加速後，再啟動魔法門8。中文版MM8Setup.Exe常會找不到3D加速硬體。第一次執行時因需要轉換一些資料檔，所以會比較慢。第二次以後可直接執行mm8.exe，但要先設定為Administrator權限，或是XP相容，否則會無法啟用3D硬體加速。

各代原本攻略，請參見６代攻略、７代攻略、８代攻略，本攻略主要列出差異的地方。

創建人物

首先要選擇的是，玩家想從那一代開始玩起。六代選安羅斯（Enroth），七代選恩塔格瑞（Antagarich），八代選賈達密（Jadame）。一般而言，八代最簡單，因為有強力的飛龍隊員；其次是六代，牧師、法師轉職快速，又能兼學光明、黑暗兩系魔法，但解除陷阱技能較弱。

接著便是要創建自己的隊伍成員。點擊Portrait左右按鍵可變更內定人物，各有其名字、種族、職業與聲音 (種族的優缺請見七代攻略)。點擊Voice左右按鍵可變更聲音，其中還包括了六代的中文語音。Class上方按鈕可變更職業，各代可變更的職業均有所限制。若點選人物畫像右上角的圖案，可以關閉職業限制，如此便可任意挑選想要的職業，包括飛龍等。不過由於涉及到轉職問題，因此最好是選在起始大陸可以轉職的職業。下方+的按鈕可增加想創建的隊員，最多為五位，也可進入遊戲後再雇用一些內定人物。

進入遊戲

進入遊戲後，第一件事便是按Esc點選控制界面，然後點選上方的額外設定：

  1. 怪物強化：內定是強化100%，如果嫌怪物太難打，可以將之關閉。
  2. 天氣效果：畫面有時會下雪或下雨，如果嫌妨礙視線，可以將之關閉。
  3. 幀數限制：限制每秒畫面幀數，以避免幀數過高導致部份電腦過熱。
  4. 怪物尋路智能增強：開啟後，怪物便不會卡在死角，很容易來到主角身邊攻擊。請自行決定是否開啟。

下方左右方向按鈕可看到更多的設定：

  1. Bolster multipliers：怪物強化時使用的各種能力係數 (如果不是很熟悉的話，不建議修改)。
  2. Keybinds：特殊按鍵設定。
  3. Interface settings：使用那一代的外觀界面，以及是否使用較小的藥劑瓶。

訓練所、旅館或商店均會隨機遇到可加入隊伍的人物（全隊最多5名），八代人物均仍在其位置待雇用。不喜歡的隊員遺散後，會回到該代的冒險者旅店等待：

  1. 6代：自由天堂
  2. 7代：塔塔力雅
  3. 8代：匕傷群島、渡鴉海濱

額外的主線

  1. 當主角經驗值達到50000（可升到10級）時，寶箱會出現訊息文件，通知主角前往某地，以遇到維爾丹，啟動跨代任務與傳送。在各代特定魔法地點，按下後再施展城市傳送，即可前往其他大陸：

    • 6代：飛龍沙漠左下綠洲的底座
    • 7代：晨昏島海中央的水系魔法顛峰公會
    • 8代：低語之森的督伊德聖圈

    各大陸的傳送目的地按遊戲已過的日數，依下列地點循環決定：

    • 6代：自由天堂、新索匹格、濃霧島、冰凍高原、布特雷格灣
    • 7代：伊拉西亞、布拉卡達沙漠、布拉卡達沙漠、晨昏島、丘陵地、土拉立安森林、戴加、塔塔力雅
    • 8代：匕傷群島、渡鴉海濱、影旋山脈、匕傷群島、渡鴉海濱、低語森林、低語森林、荒湮迴廊

    傳送後的地點可能在高空，也可能在海上，附近也許有怪物，所以要先預備好。

  2. 維爾丹要求的任務如下：

    • 協助找到遠端定位器的零件
    • 替遠端定位器零件加上魔法

    與維爾丹的交談過程中，可得到連線石，使用後即可再與維爾丹交談。但連線石使用後，必須經過6～48小時以上 (遊戲時間)，才能再次使用。

各代與原版的差異與說明

一、整體

  1. 不能輸入中文。
  2. 七代種族 (地精、精靈、矮人) 的能力優劣擴展至各代。
  3. 初始旅館即能玩魔幻牌，獲勝可得到獎金。初期可藉此獲取金錢。
  4. 接到市政廳的懸賞後，會直接在該地產生懸賞怪物，不用再到處尋找。初期亦可藉此獲取金錢。
  5. 神廟捐款後可獲得力量之時（神佑一時）法術效果，增強初期的能力。
  6. 各代對技能提升的要求條件，均同八代。例如學習術大師必須智力50以上、經商術大師必須人格50以上等。
  7. 訓練人物所需時間亦同八代，每等級8天。
  8. 678三代的裝備與神器都會出現，包括增加能力的黑藥水與白藥水，都能喝一遍，而且還有額外增加能力的藥水，因此屬性與抗性增加容易，很值得培養出一位煉金術宗師成員（隊員可更換，不必一直待在隊伍中）。
  9. 各代的魔法公會成員均共用資訊，因此在六代加入光明公會或黑暗公會後，在七代即可進入該系的魔法行會，無需選擇路線並完成初始任務。
  10. 各代的火焰台亦共用資訊，亦即+10有5個、+7有5個、+5有9個、+3有2個，因此必定有多餘的火焰台無法使用（最多只有7個屬性）。
  11. 技能達到60，若再使用增加技能裝備超過63時，實際技能值會直接減去64（即Overflow），效果反而會變糟。

二、六代

  1. 所有職業都有技能等級限制，學法術亦同。專家/大師/宗師等級一律4/7/10。
  2. 暗系／光系的學習不再做聲譽的限制，聲譽過低也不會被抓入獄（但城內警衛會開始攻擊，買賣價格也會變貴）。
  3. 必須進入過城鎮的地圖，才能施展城市傳送至該城鎮。
  4. 部份闇黑無效怪，如元素、鬼魂、牛頭人、泰坦等，因併入七代或八代相同怪物，已變成闇黑有效，可利用魔天創滅（末日審判）殺掉。但六代獨有的怪物，依然保持闇黑無效，例如阿加的怪物等。
  5. 施展末日審判殺掉平民，一樣會降低聲譽。

三、七代

  1. 原版一開始是在翡翠島參加尋寶比賽，本版若非從七代開始玩起，必須自行在週三到伊拉西亞，乘船到翡翠島參加比賽（遊戲會通知）。

四、八代

  1. 愛斯卡頓的謎題答案改成選擇題，而非問答題。
  2. 部份檔案未使用GrayFace Patch的檔案，因此一些原本已由GrayFace修正的bug會再度出現，必須特別注意，例如考瑞黑王座未交談會無法完成任務的bug。
  3. 存檔結構與原八代不同，不能使用八代存檔修改器修改人物屬性（屬於全隊的金錢、食物與日期除外）。
  4. 許多原版bug都被修正，無法使用由此延伸出來的密技。
