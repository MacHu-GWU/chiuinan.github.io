魔法門678整合版攻略 重點提示 (作者－青衫)

前言

魔法門678整合版 (Might and Magic 6,7,8 merge) 係俄國Rodril以魔法門8為基礎，加入6代與7代的內容。英文版載點：

https://www.celestialheavens.com/forum/10/16657#p369458

必須先有原始的八代英文版，然後再加入上述整合版內容。中文語言包載點：

https://github.com/might-and-magic/mm678-i18n/releases/

由於中文語言包的版本比英文版舊了很多，因此下列3個Script檔不要覆蓋，比較不會出問題：

ExtraArtifacts.lua
MonsterPathfinding.lua
Quest_CrossContinents.lua

最後請再使用魔法門678三代整合版青衫繁中更新檔，以修正一些原版已變更的資訊，並採用原繁中版名詞。不過由於各代的繁簡中文版名詞本身並不統一，因此所有名詞將依次以六代簡中、八代繁中、七代繁中的次序，儘可能符合原版，或修改成簡單易懂者為準。

第一次執行時，務必執行MM8Setup.eng.Exe (英文版) 選擇硬體加速後，再啟動魔法門8。中文版MM8Setup.Exe常會找不到3D加速硬體。第一次執行時因需要轉換一些資料檔，所以會比較慢。第二次以後可直接執行mm8.exe，但要先設定為Administrator權限，或是XP相容，否則會無法啟用3D硬體加速。

各代原本攻略，請參見６代攻略、７代攻略、８代攻略，本攻略主要列出差異的地方。

創建人物

首先要選擇的是，玩家想從那一代開始玩起。六代選安羅斯 (Enroth)，七代選恩塔格瑞 (Antagarich)，八代選賈達密 (Jadame)。一般而言，八代最簡單，因為有強力的飛龍隊員；其次是六代，牧師、法師轉職容易，又能快速兼學光明、黑暗兩系魔法，而且黑藥水購買容易。

接著便是要創建自己的隊伍成員。點擊Portrait左右按鍵可變更內定人物，各有其名字、種族、職業與聲音 (種族的優缺請見七代攻略)。點擊Voice左右按鍵可變更聲音，其中還包括了六代的中文語音。Class上方按鈕可變更職業，各代可變更的職業均有所限制。若點選人物畫像右上角的圖案，可以關閉職業限制，如此便可任意挑選想要的職業，包括飛龍等。不過由於涉及到轉職問題，因此最好是選在起始大陸可以轉職的職業。下方+的按鈕可增加想創建的隊員，最多為五位，也可進入遊戲後再雇用一些內定人物。

進入遊戲

進入遊戲後，第一件事便是按Esc點選控制界面，然後點選上方的額外設定：

  1. 怪物強化：內定是強化100%，如果嫌怪物太難打，可以將之關閉。
  2. 天氣效果：畫面有時會下雪或下雨，如果嫌妨礙視線，可以將之關閉。
  3. 幀數限制：限制每秒畫面幀數，以避免幀數過高導致部份電腦過熱。
  4. 怪物尋路智能增強：開啟後，怪物便不會卡在死角，很容易來到主角身邊攻擊。請自行決定是否開啟。

下方左右方向按鈕可看到更多的設定：

  1. Bolster multipliers：怪物強化時使用的各種能力係數 (如果不是很熟悉的話，不建議修改)。
  2. Keybinds：特殊按鍵設定。
  3. Interface settings：使用那一代的外觀界面，以及是否使用較小的藥劑瓶。

想關閉滑鼠控制視角，請參見操作按鍵。

訓練所、旅館或商店均會隨機遇到可加入隊伍的人物 (全隊最多5名)，八代人物均仍在其位置待雇用。不喜歡的隊員遺散後，會回到該代的冒險者旅店等待：

  • 6代：自由天堂
  • 7代：塔塔力雅
  • 8代：匕傷群島、渡鴉海濱

額外的主線

  1. 當主角經驗值達到50000 (可升到10級) 時，寶箱會出現訊息文件，通知主角前往某地，以遇到維爾丹，啟動跨代任務與傳送。在各代特定魔法地點，按下後再施展城市傳送，即可前往其他大陸：

    • 6代：飛龍沙漠左下綠洲的底座
    • 7代：晨昏島海中央的水系魔法顛峰公會
    • 8代：低語之森的督伊德聖圈

    各大陸的傳送目的地按遊戲已過的日數，依下列地點循環決定，玩家可利用此點，省去搭馬車或乘船前往的時間：

    • 6代：自由天堂、新索匹格、濃霧島、冰凍高原、布特雷格灣
    • 7代：伊拉西亞、布拉卡達沙漠、布拉卡達沙漠、晨昏島、丘陵地、土拉立安森林、戴加、塔塔力雅
    • 8代：匕傷群島、渡鴉海濱、影旋山脈、匕傷群島、渡鴉海濱、低語森林、低語森林、荒湮迴廊

    傳送後的地點可能在高空，也可能在海上，附近也許有怪物，所以要先預備好。

  2. 維爾丹要求的任務如下：

    • 協助製造遠端定位器 (有多個子任務)。
    • 找到失控混沌，帶入不穩定起源物質。

    與維爾丹的交談過程中，可得到連線石，使用後即可再與維爾丹交談。但連線石使用後，必須經過6～48小時以上 (遊戲時間)，才能再次使用。由於最後的跨代任務需要等待維爾丹主動通知，因此一些無關緊要的任務可以留在此期間再來處理。

各代與原版的差異與說明

一、整體

  1. 不能輸入中文。所有涉及可能輸入中文的謎題答案均改成選擇題，而非問答題，例如八代最後的愛斯卡頓謎題。
  2. 神廟捐款後可獲得力量之時 (神佑一時) 法術效果，增強初期的能力。
  3. 接到市政廳的懸賞後，會直接在該地產生懸賞怪物，不用再到處尋找 (怪物難度以主角的等級計算，如果覺得難找，用末日審判殺掉最快)。
  4. 各代對技能提升的要求條件，均同八代。例如健身術大師必須耐力50以上、學習術大師必須智力50以上、經商術大師必須人格50以上等。其他各代有差異部份均以八代為準，例如訓練人物所需時間每等級8天、木桶每次屬性+2等。
  5. 678三代的裝備與神器都會出現，包括增加能力的黑藥水與白藥水，都能喝一遍，而且還有額外增加能力的藥水，因此屬性與抗性增加容易。想要大量購買黑藥水，可到六代布萊克郡 (Blackshire)，每天更新。
  6. 各代的魔法公會成員均共用資訊，因此在六代加入光明公會或黑暗公會後，在七代即可進入該系的魔法行會，無需選擇路線並完成初始任務。
  7. 各代的火焰台亦共用資訊，亦即+10有5個、+7有5個、+5有9個、+3有2個，因此必定有多餘的火焰台無法使用 (最多只有7個屬性)。
  8. 暗系魔法公會販賣的法術書等級被弱化，除了七代深坑的夜晚公會外，其餘在六代或八代都買不到10級以上的暗系法術書 (包括魔天創滅／末日審判)。其實光系也是一樣，只不過第10級日光術不如末日審判好用，沒有也不太有影響。想修正此一不合理限制者，可修改Data\Tables\House rules.txt，將Spellbook shops裡2個9改成10。
  9. 神器武器、防具的攻防能力都略微提高。六代爆炸弓神器帕西佛 (Percival) 只有在射中敵人時才爆炸，減少傷及自身的機會 (原版為不傷自身)，且兼具物理傷害 (擊中) 與火系傷害 (爆炸)，強過原版只有物理傷害，或是其他爆炸弓只有火系傷害。
  10. 從競技場回來時，原施展的法術不會過期失效，例如攻擊加速，導致全隊變成衰弱狀態。

二、六代

  1. 所有職業都有技能等級限制，學法術亦同。專家/大師/宗師等級一律4/7/10。
  2. 暗系／光系的學習不再做聲譽的限制，聲譽過低也不會被抓入獄 (但城內警衛會開始攻擊，買賣價格也會變貴，甚至拒絕交易)。
  3. 初始旅館即能玩魔幻牌，獲勝可得到獎金。初期可藉此獲取金錢 (但七代仍需有魔幻牌才能玩)。
  4. 必須進入過城鎮的地圖，才能施展城市傳送至該城鎮。
  5. 六代除了四元素、物理傷之外，只有魔法傷害1種，七八代則再區分為心體光暗等4種，而六代怪物在轉換格式時，許多原本魔法免疫的怪物，都變成了暗系有效，於是元素、鬼魂、泰坦、金龍、牛頭人等怪物，均可利用魔天創滅／末日審判殺掉。目前無法殺掉的只剩阿加的怪物 (魔鬼水域)、魔鬼勇士、魔鬼國王 (甜水鎮)。
  6. 施展末日審判殺掉平民，一樣會降低聲譽。只要是平民陣亡，都算在主角身上，例如飛行時招引火球誤殺平民也算。
  7. 冰凍高原和幽靈沼澤的鳥身怪，施展詛咒的機率大幅降低 (和八代一樣)，不像原版六代動不動就中詛咒。
  8. 固定位置神器，不會再出現在隨機神器中，地圖重置也不會再生，如加四元素技能的莫根項鍊、加光暗技能的珍妮弗戒指等，均只能拿到一個。
  9. 銀舵前哨隱藏的霹靂寒冰經書，會變成物品充能經書 (水系第6個法術，八代和六代不同，整合版以八代為準)，其他獲得固定經書之處亦如此。同理，好用的赤焰火圈法術已不存在，只有八代類似但雞肋難用的火環術 (怒火焚身) 法術。
  10. 機器人毀滅部隊比較容易對付，因為增加了體系宗師的魔法防護可以防滅絕。施展體系宗師的魔法防護後，喝增加屬性但會有各種不良狀態的泉水，便不再受不良狀態的影響 (不會生病、死亡、滅絕等)。
  11. 魔力神眼專家NPC並非一直保持魔力神眼，而是定期重新施展，因此被解除魔法後，需要一段時間才會自動施展，不如原版好用。

三、七代

  1. 弓箭手的解除陷阱技能從最高專家級改成最高大師級，以便六代也有熟解除陷阱技能的職業隊員。
  2. 原版一開始是在翡翠島參加尋寶比賽，本版若非從七代開始玩起，必須自行在週三到伊拉西亞，乘船到翡翠島參加比賽 (遊戲會通知，不過去程需多花6天，回程則一律14天)。只要進入翡翠島，便會自動啟動七代初始任務。
  3. 選擇路線不必等城堡修復四個月後，只需完成取回獅鷲心號角任務便會觸發。取回獅鷲心號角任務也不必等城堡修復的第三個月，只需完成凱薩琳皇后和精靈王的任務即可觸發。
  4. 懸賞累積獎金包含六代的市政廳、八代的賞金獵人公會。
  5. 潛水衣只需帶在身上，不必實際著裝，便能從愛佛黎往西進到蕭爾 (因為飛龍無法著裝)。由於裝備未卸下，在蕭爾可用武器與鯊魚戰鬥 (但仍無法施法)。

四、八代

  1. 未完成擊沉雷格那艦隊任務前，先回到渡鴉海濱，會發生海盜劫掠事件。此事件在本版已被取消 (GrayFace Patch修正事項之一)。
  2. 存檔結構與原八代不同，不能使用八代存檔修改器修改人物屬性 (屬於全隊的金錢、食物與日期，或變更隊員除外)。
  3. 許多原版bug都被修正 (GrayFace Patch)，無法使用由此延伸出來的密技。例如初期偷潛入龍穴藏寶室取得優良裝備與經費、魔天創滅 (末日審判) 可殺闇系免疫怪等。

版本2023-11-05已知Bug

  1. 施展物品附魔法術時，有時會引發當機。此為較嚴重的Bug，施展前最好先存檔。若會當機，可試著將物品換到另一人身上再試看看。
  2. 技能值達到60，若再使用增加技能裝備時，並沒有將總技能值上限限制在60。因此技能值若加上裝備超過63時，實際技能值會直接減去64 (即Overflow)，效果反而會變差。
  3. 八代低語之森(Murmurwoods)：考瑞黑王座救下後，若未交談便離開，會導致永遠無法完成該任務。此Bug雖不影響過關，但會造成黑暗精靈無法升職，雖然黑暗精靈用的人極少，但影響仍不算小 (原GrayFace Patch已修正此Bug，但本版因已另行製作evt檔來處理跨代傳送，無法直接更新其evt檔)。
  4. 七代選擇黑暗路線後，仍可接下光明線大督伊德轉職任務。此Bug影響其實很小。
  5. 六代加里克的煉鐵廠(Gharik's Forge)：按下#3開關，無法開啟#4秘室門。此Bug影響很小。
  6. 神器出現的檢測程式仍有些問題，某些神器會不斷出現。
