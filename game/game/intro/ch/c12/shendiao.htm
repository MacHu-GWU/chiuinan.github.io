名稱：新神鵰俠侶 (中文版)
簡介：昱泉2001年出品的角色扮演遊戲，以3D自動視角呈現，採團隊即時方式戰鬥 [待補]。



備註：本版為2002合輯版，玩時不用換片。
保護：4CD舊版為光碟，2CD合輯版為序號：TWCA-NG51-3GDF-9DC2。
備註：

1.系統Bug非常多，時常會自動跳出遊戲，最好多存檔。
2.若進入時沒有畫面只有聲音，可嘗試將Data\video\Opening.avi刪掉，便可以正常顯示初
  始畫面了（玩時建議再設成Windows95相容）。
3.執行dxdiag，將DirectDraw加速關閉，可減少戰鬥跳出機會（但仍無法完全避免）。
4.使用秘技VECTOR會跳出時，可試著慢慢打勝來避免。反之亦然。
5.4CD版本需要換片，2CD版本不用。

密技：

1.戰鬥時輸入ARAMIS後按ENTER鍵，即可輸下列密技（皆需按Enter鍵）：
  VECTOR = 勝利
  HPFULL = 體力全滿
  MPFULL = 內力全滿
  APFULL = 真氣全滿
  LEVELUP= 等級升一級 
2.在當鋪當物品時，選一件物品（最好是高價的），然後不停按確定，每按一次就當掉一件，
  當到沒有時，仍然繼續算，於是便可一直增加大量的金錢。

修改：存檔eagle?.sav

1F94h = 楊過經驗值，不可填超過升級所需經驗，否則戰鬥完後容易當機
1F90h = 情感值，填入30 75 00 00即可到達最大的30000
2958h = 金錢

操作：用滑鼠雙擊自己可開啟選單，空白鍵可略過劇情。
