名稱：幻想西遊記 (簡中版)
簡介：金智塔2000年出品的角色扮演遊戲，以3D斜角畫面呈現，採單人即時方式戰鬥。遊戲
      的操作與界面方式幾乎完全模仿暗黑破壞神，玩家可以選擇四種角色（職業）之一做
      為主角，同樣也是開新遊戲刷新怪物，不過陣亡不會自動復活，只能讀檔重來 [待補]。



保護：無，必須放入第二片光碟才有背景音樂。
操作：

滑鼠左鍵 = 動作（行走／交談／攻擊／開箱／撿取等）
滑鼠右鍵 = 施法
Esc = 系統選單
F1 = 說明
F2 = 音效開關
F3 = 光影開關
F4 = 切換玩家合作／互殺模式
F9～F12 = 常用聊天用語
Space = 關閉所有視窗
PrintScreen = 截圖
