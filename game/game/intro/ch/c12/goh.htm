名稱：地獄門 (Gate of Hell，中文版)
簡介：智冠1999年出品的角色扮演遊戲，以3D斜角畫面呈現，團隊即時方式戰鬥 [待補]。



保護：無
備註：遊戲執行後若畫面圖案、顏色錯亂、畫面全黑，請關關DirectDraw加速，或使用DxWnd
      以視窗模式執行（但開頭動畫會無法顯示）。搭配其他輔助軟體的狀況如下：

1.dgVoodoo2：不定時當機（包括視窗模式）。
2.WineD3D：對話過場時，背景全黑（同PCem+Win98未關關DirectDraw加速的情況）。
3.DDrawCompact：顏色錯亂。
4.D3DWindower：進入遊戲時當機。
5.SoftGPU（VMware+Win98）：同WineD3D，但關關DirectDraw加速無效。

密技：以下列方式啟動遊戲：

Game.exe yoki

進入遊戲後按Enter鍵，再輸入下列字串（大小寫皆可）：

FIGHT = 無敵
MAGIC = 可學得全部的魔法，使用不扣妖力值
RAISE # = 各主角升級一次 (#為要加的級數)
GETOBJ # = 得到物品 (#為物品編號)
GOLD # = 得到金錢 (#為要加的數目)
FOG = 迷霧開關

以下為遊戲測試用，儘量不要使用：

MAP = 遊戲地圖（使用後可能會有當機狀況）
SETEVEN = 設定事件
VIEWEVEN = 觀察已設事件
VIEWMSG = 觀察訊息流向
SCREEN = 攫取螢幕畫面
BUY # = 買賣系統測試 (#為要測試的項目)

操作：

滑鼠左鍵 = 動作／移動
滑鼠右鍵 = 招式
Tab = 小地圖顯示切換
Space = 物品快捷列顯示切換
1～4 = 人物欄
S = 快速存檔
L = 快速讀檔
Esc = 系統選單
