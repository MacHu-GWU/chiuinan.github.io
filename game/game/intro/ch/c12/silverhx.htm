操作：

滑鼠左鍵 = 行走/動作
滑鼠右鍵 = 動作選單
Ctrl+滑鼠左鍵 = 攻擊（配合滑鼠移動成刺、劈、揮、砍）
Ctrl+滑鼠右鍵 = 防禦
1～3 = 選擇操控隊員
Tab = 切換操控隊員
7 = 使用最後一次使用的武器
8 = 使用最後一次使用的魔法道具
9 = 使用最後一次使用的遠程攻擊武器
0 = 使用最後一次使用的盾牌
M = 地圖
Esc = 系統選單

說明：

1.撿物品可以按著滑鼠右鍵不放，將要撿的物品圈起來即可。
2.遊戲的存檔點為固定，且均只能使用一次。
3.必須得到Amulet of Seeing後才能看到隱藏寶箱。
4.所有特技與魔法道具均可無限次使用，但用完後必須等它恢復後才能再次使用。
5.使用魔法球時，按下A鍵可使其攻擊力提高，魔法球可以用在敵人身上，也可用在自己身上。
