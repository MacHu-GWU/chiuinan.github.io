西洋棋（國際象棋）規則

一、棋盤與棋子擺置



1.白棋皇后置於白格（d1），黑棋皇后置於黑格（d8）。亦即王與王、后與后
  互相面對面，而非對稱擺置。
2.由白棋先行，棋盤右下角（h1）必須為白格。
3.每人輪流移動一次自己的棋子，先將對方的國王吃掉者勝。先行棋的人稱為
  「先手」，後行棋的人稱為「後手」。

二、各子走法與吃法

註：吃子時沒有階級大小之分，每個棋子均可吃任一軍種的棋子。

1.國王（King）：八個方向只能走一格，吃法同。有特殊的走法「入堡」。



2.皇后（Queen）：八個方向可走無限格，吃法同。為子力最強的棋子。



3.主教（Bishop）：斜角四個方向可走無限格，吃法同。



4.城堡（Rook）：直線四個方向可走無限格，吃法同。



5.騎士（Knight）：直走兩格，再往旁邊走一格，可跨越其他棋子，吃法同。



6.士兵（Pawn）：走法與吃法不同，左邊為走法，右邊為吃法。

 

(1)未曾移動過的士兵可選擇前走一格或兩格，否則只能前走一格，不能後退。
   當士兵走到棋盤底端邊界時，可以進行升變。
(2)士兵只能吃前方斜角的子，無法吃正前方的子。有特殊的吃法「吃過路兵」。

三、特殊規則

1.入堡：把國王向城堡方向橫移兩格，再把城堡直接移到國王的另一側相鄰格。
  但有下列限制條件：
  (1)國王、城堡都必須是第一次移動。
  (2)國王和城堡之間不能有其它棋子。
  (3)國王所在位置與橫移的兩格（共三格）都不能處於受對方攻擊的狀態。
2.升變：當士兵抵達棋盤的另一端時，它可以變成任何一種棋子（通常為皇后）。
3.吃過路兵：如果一個士兵第一次移動時前進兩格，旁邊剛好有對手的士兵，
  亦即直接跳過對手士兵可以吃子的棋格，則對手士兵可以選擇吃掉這個士兵，
  且必須立即施行，否則失效。
4.和局條件：
  (1)棋盤上沒有足夠的棋子去將死對方
  (2)同一位玩家同樣的局面重複三次。
  (3)連續50步中，沒有玩家移動士兵，也沒有吃子情形。

四、開局基本概念

1.控制棋盤的中央地帶：指最中央的四格。由於處在中央地帶，能威脅的格子
  最多，所以要搶先佔領，或是威脅該處，避免對手佔領。
2.第一步棋的選擇
  (1)從國王前的士兵開始：可同時打通皇后與主教的通路，表面上國王似乎
     容易遭受攻擊，但其實不然。
  (2)從皇后前的士兵開始。
  (3)從騎士開始。
  以此形成【雙王前兵開局】、【單王前兵開局】、【雙后前兵開局】等多種
  開局法。
3.儘快出子，使自己的子力能迅速到達進攻或防禦的位置。以騎士和主教為優
  先，皇后應做後援，而非先鋒。
4.建立起好的兵形，使各個士兵之間能互相呼應，以便進攻和防禦。吃子時亦
  應先選擇非士兵，避免破壞兵形。
5.國王要儘早移往較安全的位置（通常是角落，採用入堡移動法）。

五、中殘局基本概念

1.設法「將死」對方國王。
2.殲滅對方棋子。兌子時，要注意敵我雙方損失的子力是否划算。各子的重要
  性為：
  (1)士兵1分
  (2)騎士3分
  (3)主教3分
  (4)城堡5分
  (5)皇后9分
