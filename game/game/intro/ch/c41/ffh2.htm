名稱：風火跳棋2 (中文版)
簡介：倚天1991年出品的跳棋遊戲。遊戲提供了兩人或三人的10子跳棋，各人物都可選擇由
      電腦或玩家控制，算是一款還可以的跳棋遊戲。

保護：無。
備註：FHCII.EXE執行參數

/R = 重新設定設備
/A = 使用Adlib
/P = 使用PC喇叭
/K = 忽略滑鼠
/MGA或/HERC = 單色螢幕

操作：

F = 強迫電腦停止思考，立即下子
Esc = 停止電腦提示一步的思考

Q = 結束遊戲
N = 重來一局
S = 儲存棋局
L = 取出棋局
P = 列印棋步
E = 修改棋局
O = 音效開關
B = 倒退一步
H = 提示一步
G = 歷史重演，將下過的棋步從頭演示一遍，以檢討得失
      T = 停止演示
      其他鍵 = 暫停／繼續演示
M = 查看目錄
+- = 調整電腦棋力，共分三個等級1～3，預設為1
K = 老闆來了，暫時跳出遊戲，再按任何一鍵返回
A = 設定電腦思考時間（10～2000秒）
