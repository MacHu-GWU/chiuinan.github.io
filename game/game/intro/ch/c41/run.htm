名稱：賽跑棋 (中文版)
簡介：船塢資訊1990年出品的棋類遊戲，附加在三角跳棋磁片裡。遊戲的目標是將己方所有
      棋子移到另一邊底部，移動的步數由骰子決定。規則很簡單，雖然勝負有很大一部份
      是靠運氣，但電腦AI明顯不行，因此並不是很耐玩。



保護：無
限制：必須單色螢幕，DosBox請修改conf檔machine=hercules。
說明：

1.先按Space鍵擲骰，決定移動的步數（顯示在畫面下方）。
2.可自行決定移動那一個棋子（我方為上面白子），一次必須移動前面所擲骰子的步數。
3.接近邊界的棋子，不能移動超過其最大剩餘步數。如果無子可移動，必須按Esc略過。
4.率先移動所有棋子到邊界的人獲勝。
5.最好一開始先將一顆棋子移動到接近邊界（剩下1～6步），若剛好擲出可到邊界的點數，
  則立即移到邊界，否則便移動其他棋子繼續前進到接近邊界。
