名稱：三角跳棋 (中文版)
簡介：船塢資訊1990年出品的棋類遊戲，雖名為跳棋，但其實是孔明棋的變形。遊戲有15顆
      棋子擺成正三角形，玩家第一手可以拿走任意一顆棋子，之後便以跳棋的形式，將跳
      過的棋子拿走，目標便是讓棋盤上最後只剩下一顆棋子，且位置在三角形最底層中央。
      遊戲雖有趣，但並非很耐玩。如果不知如何走，遊戲中也可傾印各種走法，算是比較
      貼心的一點。

保護：無
限制：必須單色螢幕，DosBox請修改conf檔machine=hercules。
操作：

上下左右 = 移動游標
Enter = 確定選擇棋子，進行移動並吃子
BackSpace = 退回到上一手
Esc = 結束遊戲，回到DOS
F1 = 印出所有可能的成功走法
F2 = 重新再玩
F3 = 顯示玩家最近25個成功走法
F4 = 請電腦幫忙下棋，並搜尋、顯示可能的走法
       Space = 加速搜尋
       Esc = 結束搜尋
       其他鍵 = 暫停搜尋
F5 = 切換聲音
F6 = 切換動畫
F7 = 顯示遊戲規則

說明：

1.第一手可拿走任何一顆棋子。
2.第二手後以跳棋的方式拿走被跳過的棋子。
3.最後必須只剩一個棋子，且位置在15。

由 1開始，有 6280種成功走法。
由 5開始，有  538種成功走法。
由 7開始，有 3140種成功走法。
由10開始，有 3140種成功走法。
由15開始，有19870種成功走法。

其餘無成功走法。
