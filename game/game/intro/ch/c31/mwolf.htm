名稱：烈焰鋼狼傳 (中文版)
簡介：智冠1996年出品的策略角色扮演遊戲 [待補]。



備註：本版為光碟版（有開頭動畫）
保護：破解碼：
      TESTIMG.EXE
      1.磁片版（數字密碼）
        0F 85 C2 FE FF FF B8 01
        90 90 90 90 90 90 -- --
      2.光碟版
        0F 84 67 04 00 00
        -- -- 00 00 -- --
        0F 84 3D 04 00 00
        -- -- 00 00 -- --
備註：DosBox執行時，光碟版必須將SETUP.CFG設成CD=0，否則會當機。主要是因為遊戲使用
      0:0做為int 2Fh的Device Driver Request Header資料位址，但這個位址在DosBox裡
      會被視為錯誤略過。於是遊戲一開始在取得音軌資訊時，因為DosBox沒有成功執行，
      而應填入的位址仍為亂碼，會使得遊戲誤認有大量的音軌，將之全部讀入，因為過長
      而覆蓋到其他資料，使得後面大亂而當機。這部份只能修正DosBox的原始程式
      cdrom_image.cpp，將函數MSCDEX_Interrupt_Handler一開始下列的程式刪掉：

if (curReqheaderPtr==0) {
   MSCDEX_LOG("MSCDEX: invalid call to interrupt handler");
   return CBRET_NONE;
}

檔案：mwolf-hd.rar = 磁片版
密技：

1.不死大法：當遊戲動畫有打開時，只要在被子彈打到前按Esc跳過動畫即不會受傷，可利用
  此點避免友軍或較弱的同伴被打死。也可以讓敵人不被友軍打到，留下來增加己方經驗值。
2.跳過一回合：在戰場畫面時，選擇任一位我方未行動過的機兵，打開單位指令輸入ming.，
  可跳過一回合。
3.敵人全滅：在戰場畫面時，選擇任一位我方未行動過的機兵，打開單位指令輸入mini.，即
  可讓戰場上所有敵人全滅。
4.迅速升級：在戰場畫面時，打開整體指令輸入mini.，可使我方所有單位升至99級。
5.可購買所有物品：在整備畫面、商人出現時，在整備選項上輸入mini.（輸入要夠快），
  金錢增至一千萬，且所有可購買單位會全出現（包含特殊機兵）。
6.無限移動次數：選中要移動的角色（已經移動過的），按Enter顯示屬性，再按Enter退到
  待命狀態，按下S鍵，向沒有陰影的地方移動後，再按下Esc鍵，此回合便可多次移動。
7.無限移動次數：將方格移到人物的下面，按下S鍵，會出現一格可移動，移動完後就能無限
  行動。
8.控制敵軍和友軍：用無限移動次數法移動一個角色數次，然後就可移動敵軍和友軍。否則
  遊戲會退回DOS下。
9.提高初始等級：在有等級比較高的戰場記錄時回到遊戲主選單，再選擇重新開始遊戲，會
  發現戰士等級會跟等級比較高的戰場上記錄的等級一模一樣。

操作：

一、地圖游標及選單

↑ 或 [8]           往上移動
↓ 或 [2]           往下移動
← 或 [4]           往左移動
→ 或 [6]           往右移動
[Space] 或 [Enter]  確定鍵
[Esc] 或 [NumLock]  叫出整體指令選單或跳離各種選單畫面

二、熱鍵

[F1]    找尋（將游標移至尚未行動的部隊身上，以編號為順序）
[F2]    打開部隊資料畫面
[F3]    打開縮小地圖視窗
[F4]    顯示過關條件
[F5]    打開機能選單
[F6]    結束本回合
[F9]    存檔，並跳回主選單
[F10]   讀取存檔
[Ins]   地形資料
[~]     抓圖，所抓取的圖形會存成PCX檔
