密技：先同時按下Ctrl+S+H啟動密技：

滑鼠左鍵 = 可選擇敵我任一隊伍
N = 本回合重新開始
= = 將領升級
F5 = 直接勝利過關
F6 = 我方失敗
F7 = 直接結局
F8 = 關閉密技
/ = 顯示游標所在位置的相關資訊

操作：

Space = 移至尚未行動的我方隊伍
F1 = 全體休息
F9 = 自動進行（無法停止）
S = 快速存檔
L = 快速讀檔
V = 查看地圖
Esc = 系統選單
滑鼠左鍵 = 選擇我方隊伍
滑鼠右鍵 = 整體戰鬥命令
