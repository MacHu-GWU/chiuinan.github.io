      sg1-steam.rar = Steam版
密技：

SANGO.EXE後面可加參數：

-DM        啟用密技
-AUTO      由電腦AI代替玩家出戰、徵兵，金錢與預備兵只增不減
-AIOFF     關閉敵軍電腦AI，使敵軍不能動作
-BATTLE #  直接進入戰場#，選擇退兵後可獲得金錢9150元
-MAPMODE # 同上，但只得金錢
-INFO      在畫面左上顯示記憶體使用狀況，進入戰鬥會顯示所在的座標
-SPEED     增加武將技速度
-FIXSPEED  增加武將技速度
-FULLSCREEN
-DEBUGALL  在大地圖模式可任意控制敵方出征及徵兵，在戰鬥模式時
             連按數次F1 F2 F3，則敵方使用武將技
             連按數次F5 F6 F7，則我方使用武將技
             F4配合上方按鍵，變換武將技

啟用密技後：

1.內政模式
  執行搜索時，會將人才一一搜出，直到此地無人為止
2.大地圖模式
  F1：大地圖的日期停止增加（停在當時的月份不動）
  F2：直接進入一年的最後一刻，如沒有使用F1的功能時，便會進入內政
  F3：城內增加10000金
  F4：城內的預備兵補滿
  F5：所有的武將等級加一級
3.作戰模式
  F8：武將技的氣會馬上集滿
  F9：作戰時間=999
  F10：作戰時間=0

操作：

A,S,D,Z,X,C 變換視角
R,U,T,Y     變換人物大小

說明：

遊戲破關條件為西元280年以前，佔領所有城池，統一中國。

雖然本遊戲一開始時並不能選擇時期，且只可以選擇一位君主開始遊戲，但若是全破過一次
後，再次選擇開始新遊戲時便可以選擇時期（劇本），每個時期亦有不同的君主可以選擇。
