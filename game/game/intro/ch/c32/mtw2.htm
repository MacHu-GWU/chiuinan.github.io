名稱：中世紀2：全軍破敵／全面戰爭 (Medieval 2: Total War，簡中／英文版)
簡介：Creative Assembly 2006年出品的回合戰略遊戲，由Sega代理發行，為全軍破敵系列
      的第四部作品。2007年推出「王國（Kingdoms）」資料片，以及結合兩者的「黃金版
      （Gold Edition）」 [待補]。



保護：光碟SafeDisc+序號（簡中版：FG5H-D43X-X7YB-JTTY-KA6S）
備註：執行時需要D3DX9_30.DLL，虛擬機無法執行，請使用實體機。
備註：必須執行[中世纪II全面战争之王国.exe]才有中文，繁中系統若無法執行，可將之改
      名為英文（包括同名的cdd檔）。或是以下列方式直接執行：

1.主程式：medieval2.exe --io.file_first
2.王國資料片：kingdoms.exe --io.file_first --features.mod=mods/MOD名稱
  MOD名稱可為：americas、british_isles、crusades、teutonic

檔案：nocd.rar = 免光碟檔（主程式1.3版，王國資料片1.5版）
      漢之殤.rar = 漢之殤MOD
資訊：參數檔medieval2.preference.cfg

[io]
file_first = true       ; 使用外掛text文字檔
[features]
editor = true           ; 啟動Battle Editor
mod = mod所在目錄       ; 啟用MOD
[misc]
unlock_campaign = true  ; 開放所有章節遊戲
[video]
windowed = true         ; 視窗模式
[video]
movies = false          ; 不播放影片
[game]
unlimited_men_on_battlefield = 1    ; 進入戰場不限制人數
