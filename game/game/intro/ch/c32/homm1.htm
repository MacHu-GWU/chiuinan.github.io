名稱：魔法門之英雄無敵1：戰略任務 (Heroes of Might and Magic: A Strategic Quest，中文/英文版)
簡介：New World Computing 1995年出品的戰略遊戲，由智冠中文化並代理發行 [待補]。





備註：本版已修正城堡塔樓射箭會引發當機的問題。
保護：光碟。英文版必須模擬光碟，並將LABEL設成HEROES_SIM：
        mount d [任一目錄] -t cdrom -label HEROES_SIM
      中文版則模擬的光碟上面必須有下列檔案（內容不管）：
        HEROES\SOUND\HEROES01.62S
檔案：HoMM1_CHT_Patch.rar = 繁中更新檔
      hpatch12.rar = 英文DOS 1.2更新檔
      hw9511.rar = 英文Win95 1.1更新檔
密技：遊戲中輸入以下數字：

695100 強迫關卡勝利
784510 強迫關卡失敗
876102 給目前正在動作的英雄所有法術（必須要有魔法書）
200978 提升目前正在動作的英雄800經驗值
528921 給目前正在動作的英雄10雙獅鷲
903376 增加1000黃金及各種資源10
101495 顯示所有地圖

操作：

1.冒險時

滑鼠右鍵 = 顯示說明訊息
方向鍵 = 移動英雄
Ctrl+方向鍵 = 捲動地圖
H = 顯示下個英雄的位置
T = 顯示下個城堡／村莊位置
Tab = 換下個有行動點數的英雄
M = 自動移動有指定路徑的英雄
K = 王國概觀
E = 回合結束
A = 冒險選項
      V = 觀看世界地圖
      P = 觀看石塔謎題
      C = 施法
      D = 挖掘無上寶物，必須花費一回合所有行動點數
G = 遊戲選項
      N = 新遊戲
      L = 載入進度
      S = 儲存進度
      Q = 離開遊戲
      I = 關卡簡介
Enter = 是／完成／接受
Esc = 否／跳出／放棄

2.戰鬥時

空白鍵 = 目前部隊按兵不動一回合
A = 自動戰鬥
C = 施法
T = 觀看部隊資料
