名稱：三國風雲1 (中文版)
簡介：世紀縱橫1996年出品的即時戰術遊戲，由智冠代理發行。1998年再推出「風雲變色」
      資料片，增加劇本以及地圖編輯器。2003年由冠捷以資料片為基礎發行「旗艦版」，
      取消了動畫與地圖編輯器 [待補]。



備註：本版為旗艦版
保護：光碟，旗艦版破解碼（必須放入光碟才有背景音樂）：
      Sango95.exe
        74 2D 8A D8 A1 58
        EB 50 -- -- -- --
備註：本遊戲有三個版本的光碟，其執行方式如下：

1.1996年DOS版：使用DosBox執行的問題很多，必須使用DosBox-X採DOS開機方式勉強可以執
  行，建議在虛擬機+XP或Win9x直接以純DOS模式執行。
2.1998年Win95風雲變色資料片：必須和DOS版安裝在同一目錄才能正常執行，XP以上系統必
  須設定SANGO95.EXE與SANEDT95.EXE為Win9x相容。
3.2003年旗艦版：仍必須設定Sango95.exe為Win9x相容，否則在XP以上系統執行仍有問題。
