名稱：異形地球 (Alien Earth，中文/英文版)
簡介：Beam Software 1998年出品的動作冒險遊戲，也帶了一部份的角色扮演遊戲 [待補]。



保護：光碟，破解法如下：

1.將光碟中Alien目錄（整個目錄）拷貝到安裝目錄中。
2.修改registry：

[HKEY_LOCAL_MACHINE\Software\Alien Earth]
"TargetDrive"=".\\"

備註：若安裝時出現"請先移除先前的異形地球版本，才能重新安裝遊戲"的訊息時，請先刪除：
      HKEY_LOCAL_MACHINE\Software\Alien Earth的registry。
操作：

對地上物品按下滑鼠右鍵，可出現處理選單
對敵人按下滑鼠左鍵，可進行攻擊
