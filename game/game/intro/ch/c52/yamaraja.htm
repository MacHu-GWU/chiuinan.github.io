名稱：般若魔界 (中文版)
簡介：昱泉1998年出品的動作冒險遊戲，由第三波代理發行 [待補]。



備註：本版為1.10版（黃金版）
保護：光碟，破解碼（必須放入光碟才有背景音樂）：
      Yama.exe
        85 C0 74 1B 85 C9 75 17  (1.00版)
        -- -- 90 90 -- -- 90 90
        85 C0 74 21 85 C9 75 1D  (1.01版或1.02版)
        -- -- 90 90 -- -- 90 90
        00 85 C0 74 4F A1        (1.10版)
        -- -- -- 90 90 --
      必須將必要的光碟檔案拷貝到安裝目錄裡（參見YmPath.ini），當要求放入光碟時，
      直接按Enter即可。
備註：安裝時若Ymsetup.exe當死，用工作管理員將它結束掉，即可繼續安裝，但安裝完後要
      自行修改YmPath.ini裡的目錄（有些要指向安裝目錄）。
備註：執行時，後面加參數L，表示使用256色模式，否則為65536色模式。
備註：儘量將YmPath.ini裡需要的目錄檔案拷到安裝目錄裡，執行比較不會出問題。256色模
      式拷貝第一片，65536色模式拷貝第二片。
備註：虛擬機+XP若無法直接執行，請使用DxWnd。不建議使用Win64系統，相容性問題多。
檔案：YAMA101.rar = 1.01更新檔（威力加強版）
      YAMA102.rar = 1.02更新檔（含1.01更新檔內容）
      YAMA_GOLD.rar = 黃金版與1.02版差異檔
密技：更新至威力加強版後，遊戲中按下F3，再輸入：

naraka 啟用生命恢復密技，之後按L键便可恢復生命和能量至全滿
ken    獲得人劍
lama   獲得天弓 
taka   獲得阿修羅匕首
koma   獲得無間珠

操作：

上下左右鍵 = 移動
按住上鍵 = 推動物品
Shift = 備戰開關
Space = 蹲下取物
Z = 跑步／游泳
Ctrl = 跳躍
Ctrl+上鍵 = 上段攻擊
Ctrl+左右鍵 = 中段攻擊
Ctrl+下鍵 = 下段攻擊
Alt = 蹲下／閃躲
Ctrl+Alt = 必殺技
1～8 = 選用物品欄中物品
Tab = 視窗上下界面顯示開關

必殺技使用條件：

1.必須滿血。
2.武器等級3以上。
3.雪使用人劍、龍使用天弓、康拉德使用阿修羅匕首。
4.按住Ctrl+Alt聚氣，等血的綠血消失再放掉即可。
