名稱：洛克人Zero 3 (Rockman Zero 3／Mega Man Zero 3，中文／英文版)
簡介：CAPCOM 2004年出品的動作遊戲，2020年經由Steam移植至Windows，收錄在「Zero&ZX
      雙雄合集（Legacy Collection）」中 [待補]。
 
備註：本版為Steam合集版，請參見洛克人Zero 1。
保護：Steam
備註：Steam合集版劇情文字仍為英文，並非中文。
操作：

Space = 確定／攻擊（按久一點為集氣）
A = 取消／跳躍／蹬牆
D = 衝刺
C = 使用次要武器（按著C再按Space攻擊）
上鍵 = 交談／爬梯／開門
Esc = 內部選單／略過劇情
Tab = 外部選單
