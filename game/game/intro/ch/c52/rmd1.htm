名稱：洛克人DASH：鋼鐵之心／鋼之冒險心 (Rockman DASH／Mega Man DASH，中文版)
簡介：CAPCOM 1997年出品的動作遊戲，屬遊戲系列中的「DASH系列／傳奇系列」分支。2000
      年由第三波中文化並代理發行 [待補]。



保護：不明
操作：

上下鍵 = 前進後退
左右鍵 = 平移
Del/PgDn = 轉身
PgUp = 拉近
Enter = 開始／功能選單
Z或Esc = 取消／右擊
C = 發射／左擊
X或Space = 確定／跳躍
V = 開啟／交談
右Shift = 地圖
F8 = 全螢幕／視窗化切換
F9 = 離開遊戲／結束遊戲
F10 = 暫停遊戲
F11 = 顯示fps
