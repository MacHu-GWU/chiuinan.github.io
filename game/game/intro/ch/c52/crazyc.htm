名稱：蜘蛛超人 (Crazy Climber，中文版)
簡介：Nichibutsu 1998年出品的動作遊戲，由華義中文化並代理發行。遊戲型態改編自舊式
      街機「大金剛」，玩家必須克服各種敵人與障礙，爬到樓層頂端。遊戲共有七關，各
      關均可無限次挑戰。只是遊戲頗有難度，隻數很少又沒有接關的功能，想要過關可說
      非常困難。遊戲最大的問題點在於，陣亡後復活的人物沒有回到安全的地帶，常常會
      造成復活後立刻陣亡，更加深了遊戲的困難度。不過遊戲還是有一定的娛樂性，值得
      玩家來挑戰看看。



保護：光碟，破解碼（必須放入光碟才有背景音樂）：

操作：（可在CONFIGURATION裡設定）

AD = 左右
2+8 = 往上爬（Easy級只需按2）
Space = 使用物品
Enter = 投幣
Esc = 離開

說明：

1.建議選擇速度較快的主角。
2.掉下的輕物，只要雙手都有抓住，被擊中便不會掉下。但有些重物，只要被擊中就會掉下。
3.物品最多只能有一個。用途為：
    炸藥：清除畫面上的敵人與障礙物。
    時鐘：所有敵人與障礙物均停止一段時間不動。
4.第七關第二個頭要設法接住它丟出的炸藥並使用，約炸四次便能去除。
