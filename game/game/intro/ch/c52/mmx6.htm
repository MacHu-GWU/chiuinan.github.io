名稱：洛克人X6 (Rockman X 6／Mega Man X 6，中文版)
簡介：CAPCOM 2001年出品的動作遊戲，2003年移植至Windows。2018年經由Steam重新移植，
      收錄在「遺產合集2（Legacy Collection 2）」中 [待補]。



備註：本版為原始繁中光碟版（未破解，需放入光碟），Steam合集版請參見洛克人X7
保護：原始繁中光碟版為光碟SafeDisc 2.90
備註：原始繁中光碟版VirtualBox無法執行，虛擬機請使用VMware。
備註：Steam合集版（請參見洛克人X7）劇情文字仍為英文，並非中文。
操作：

一、原始光碟版配置

Enter = 開始/略過劇情
C = 射擊（按久一點為集氣）
V或Esc = 跳躍
Space = 衝刺
X = 特殊武器
S/F = 變更武器
B = 超強力攻擊
F3 = 降低速度
F4 = 加快速度
F5 = Frame Skip開關
F6 = 降低亮度
F7 = 內定亮度
F8 = 提高亮度
F9 = 離開遊戲（主選單為結束遊戲）
