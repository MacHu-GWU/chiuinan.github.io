名稱：洛克人9：野心的復活 (Rockman 9／Mega Man 9，中文／英文版)
簡介：CAPCOM 2008年出品的動作遊戲，2017年經由Steam移植至Windows，收錄在「傳奇合輯2
      （Legacy Collection 2）」裡 [待補]。
 
備註：本版為傳奇合輯2，請參見洛克人7
保護：Steam
操作：

Enter = 開始/選擇
Z = 攻擊
X = 跳躍
Shift = 連射
W = 選擇武器
Tab = 內部功能選單（原始遊戲的功能選單）
~ = 外部功能選單（整合程式的功能選單）

說明：可選關卡：

1.Concrete Man
2.Jewel Man
3.Hornet Man
4.Tornado Man
5.Plug Man
6.Galaxy Man
7.Magma Man
8.Splash Woman
