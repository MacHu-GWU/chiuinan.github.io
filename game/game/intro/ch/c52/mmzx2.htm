名稱：洛克人ZX降臨 (Rockman ZX ADVENT／Mega Man ZX ADVENT，中文／英文版)
簡介：CAPCOM 2007年出品的動作遊戲，2020年經由Steam移植至Windows，收錄在「Zero&ZX
      雙雄合集（Legacy Collection）」中 [待補]。
 
備註：本版為Steam合集版，請參見洛克人Zero 1。
保護：Steam
備註：Steam合集版劇情文字仍為英文，並非中文。
操作：

X = 主要攻擊
C = 次要攻擊
Space = 確定／跳躍／蹬牆
A = 取消／Giga Crush
D = 衝刺
S = 合體變身
F = 開關副畫面
Z或Enter = 以副畫面游標進行操作
MK,. = 移動副畫面游標
Esc = 內部選單／略過劇情
Tab = 外部選單
