名稱：洛克人X3 (Rockman X 3／Mega Man X 3，中文／英文版)
簡介：CAPCOM 1995年出品的動作遊戲，1997年移植至Windows，2018年經由Steam重新移植，
      收錄在「遺產合集（Legacy Collection）」中 [待補]。
 
備註：本版為Steam合集版，參見洛克人X1。
保護：原始光碟版為光碟，破解碼（需完全安裝，有放入光碟才有背景音樂）：
      MMX3.EXE
        74 16 6A 02
        -- 14 -- --
檔案：mmx3-cd.rar = 原始英文光碟版
操作：

一、原始光碟版配置

Z或Space = 射擊（按久一點為集氣）
X或Esc = 跳躍
C = 衝刺
S/D = 變更武器
A = 選擇武器（再按下D可選擇是否呼叫傑洛）
F4 = 全螢幕/視窗切換
F7 = 影像縮放
F9 = 離開遊戲（主選單為結束遊戲）
F10 = 遊戲暫停
