名稱：幻獸都市：魔法大地 (中文版)
簡介：天堂鳥1993年出品的動作遊戲。玩家必須搥打方塊以獲得必要過關物品與魔法，遊戲
      共有八大關卡，每大關均有三小關以及最後的魔王關。雖然遊戲基本上還頗好玩，但
      因無法隨時查閱主角狀態與擁有的魔法，加上後面的關卡愈來愈難，遊戲存檔卻無法
      儲存關卡現況，因此頗考驗玩家的耐心，有點影響遊戲的娛樂性。
 
保護：數字密碼，破解碼：
      HAMMER.EXE
        75 24 C7 06 9A
        -- 00 -- -- --
備註：正常版請執行PLAY.BAT，不死版請執行UPLAY.BAT。
備註：進入遊戲若英數字顯示不正常，可能檔案為唯讀屬性，請將之取消。
備註：電腦速度不可太快，否則可能會偵測不到音效卡（或使用較新的SVN-Daum DosBox）。
備註：INSTALL.EXE變更設定後，儲存的設定檔GAME.CFG是放在根目錄裡，而非安裝目錄。
修改：

1.隻數不減
  HAMMER.EXE
    FE 0E C5 78  (共5個)
    90 90 90 90
2.法力不減
  HAMMER.EXE
    8A 87 B9 78 04 FB
    -- -- -- -- 90 90
    8A 87 B9 78 04 F6  (共4個)
    -- -- -- -- 90 90
3.修正第七大關最後魔王前被一排洞穴擋住，無法前進的問題
  HAMMER7.TBL位置143Ch，將2Eh改成22h

修改：存檔HAMMER.SAV（以下以第一存檔為準）

3Dh～3Eh：經驗值
3Fh：魔法等級
41h～4Ch：各魔法法力（00表尚未學會）
4Eh：第幾大關（00～07）

資訊：各關的方塊資訊檔HAMMER?.TBL

348h byte一組，最前面14x2x10=280=118h為方塊定義，每個方塊2 byte，共計14x10（第一
行不用）。各方塊第一byte為形態定義值，如2Eh為洞穴。

操作：

方向鍵 = 移動／閃躲
Space = 敲磚塊／攻擊
Ctrl = 切換法術
Alt = 切換防具
Esc = 系統選單（暫停）

說明：

1.遊戲共有八大關卡，每大關卡再分三小關卡與最後的魔王戰。
2.打魔王時需施展法術，各法術均需要法力才能施展（絕大部份每次施展均需10法力），可
  看右下角圓石上方的條數（目前法術的剩餘法力），每一完整條為10，全數用盡即無法再
  施展該法術。必須陣亡才能恢復法力。
3.某些法術攻擊魔王反而會幫補血，必須特別注意施法後敵方的生命數字增減。
4.可透過打死小關的怪物獲得經驗值，以提升魔法等級，增加法力。愈後面的關卡，小關的
  怪物愈會追著主角跑。　
5.第一關一定要練到等級2以上，否則法力不足以施展攻擊法術，面對魔王會完全無法過關。
  練到等級3才有足夠法力不陣亡殺死魔王。
6.魔法必須在小關裡打磚塊獲得（共12種）。
7.陣亡的怪物會在過一段時間後原地復活。若地圖上有固定洞穴，可將怪物推向洞穴中，如
  此怪物一復活便立死，可無限增加經驗值。
8.儲存進度係以大關為準，讀取後會回到該大關的第一小關，同時魔法等級也會被降級。
