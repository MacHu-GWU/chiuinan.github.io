名稱：機甲戰士 (Hamos: The Green Chariot，中文版／韓文版)
簡介：韓國I'll Soft 1999年出品的動作射擊遊戲，由旭力亞中文化並代理發行 [待補]。



備註：本版為繁中版，光碟為韓文版
保護：光碟，破解碼（安裝目錄必須在C磁碟才能執行）：
      Hamos.exe
        83 F8 05 75 07
        -- -- -- 90 90
      要將光碟上的檔案拷到安裝目錄裡。
備註：虛擬機若無法執行，可使用DxWnd或D3DWindower執行。
操作：

滑鼠左鍵 = 劇情模式移動人物
滑鼠右鍵 = 劇情模式移動地圖（按著拖曳）
上下左右鍵 = 戰鬥模式移動人物
1～7 = 變更武器
8～0 = 變更炸彈
Ctrl = 攻擊／射擊
Alt = 射擊
Tab = 更換武器
Esc = 系統選單
