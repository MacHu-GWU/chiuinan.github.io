名稱：洛克人X4 (Rockman X 4／Mega Man X 4，中文／英文版)
簡介：CAPCOM 1997年出品的動作遊戲，1998年移植至Windows，2018年經由Steam重新移植，
      收錄在「遺產合集（Legacy Collection）」中 [待補]。
 
備註：本版為Steam合集版，參見洛克人X1。
保護：原始光碟版為光碟，破解碼（需完全安裝）：
      RMX4.exe
        75 45 8B 1D
        EB -- -- --
檔案：mmx4-cd.rar = 原始英文光碟版
操作：

一、原始光碟版配置

C或V或Space = 射擊（按久一點為集氣）
X或Esc = 跳躍
Z = 衝刺
S/A = 變更武器
F4 = 全螢幕/視窗切換
F5 = 切換螢幕解析度（320x240、640x480、800x600）
F6 = 切換顏色數（8bit、16bit、32bit）
F7 = 影像縮放
F9 = 離開遊戲（主選單為結束遊戲）
