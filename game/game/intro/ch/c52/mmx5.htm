名稱：洛克人X5 (Rockman X 5／Mega Man X 5，中文版)
簡介：CAPCOM 2000年出品的動作遊戲，2001年由大新資訊中文化，並由第三波發行。2018年
      經由Steam重新移植，收錄在「遺產合集2（Legacy Collection 2）」中 [待補]。
 
備註：本版為原始繁中光碟版，Steam合集版請參見洛克人X7
保護：原始光碟版為光碟，破解碼不明
備註：原始光碟版VirtualBox無法執行，虛擬機請使用VMware。
備註：Steam合集版（請參見洛克人X7）劇情文字仍為英文，並非中文。
操作：

一、原始光碟版配置

Enter = 開始
C = 射擊（按久一點為集氣）
V或Esc = 跳躍
X或Space = 衝刺
D = 特殊武器
F7 = 變更影像顯示方式
F8 = 全螢幕/視窗切換
F9 = 離開遊戲（主選單為結束遊戲）
