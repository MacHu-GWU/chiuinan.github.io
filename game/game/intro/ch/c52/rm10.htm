名稱：洛克人10：來自宇宙的威脅 (Rockman 10／Mega Man 10，中文／英文版)
簡介：CAPCOM 2010年出品的動作遊戲，2017年經由Steam移植至Windows，收錄在「傳奇合輯2
      （Legacy Collection 2）」裡 [待補]。
 
備註：本版為傳奇合輯2，請參見洛克人7
保護：Steam
操作：

Enter = 開始/選擇
Z = 攻擊
X = 跳躍
Shift = 連射
W = 選擇武器
A/D = 變更武器
Tab = 內部功能選單（原始遊戲的功能選單）
~ = 外部功能選單（整合程式的功能選單）

說明：可選關卡：

1.Blade Man
2.Solar Man
3.Sheep Man
4.Chill Man
5.Commando Man
6.Nitro Man
7.Strike Man
8.Pump Woman
