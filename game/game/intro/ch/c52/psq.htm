名稱：激戰足球學園 (Phantasy Soccer Q，中文版)
簡介：宇峻2003年出品的足球運動遊戲 [待補]。



保護：光碟StarForce 3.03+序號（0901148226907040281214207）
限制：由於StarForce保護的關係，只能在XP上執行。
檔案：PSQPatch102.rar = 1.02更新檔（需輸入序號）
操作：

F5 = 暫停

1.控球人員
  X = 傳球（按第一次集氣以控制力道，按第二次傳出）
  C = 射門
  Enter = 必殺射門（需有射門必殺技且士氣100時）
2.守球人員
  V = 剷球
  X或0 = 切換守球人員
3.守門員
  方向鍵 = 連按可增加接球力
  Enter = 必殺守門（需有守門必殺技且士氣100時）
  C = 傳給後衛（拿到球後）
  V = 傳給前鋒（拿到球後）
