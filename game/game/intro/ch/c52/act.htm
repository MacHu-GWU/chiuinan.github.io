名稱：格鬥拳王 (中文版)
簡介：大宇1994年出品的動作打鬥遊戲，以橫向捲軸前進，和西遊記相當類似，只是無法選
      擇人物，而且也只能單打。遊戲一共有５關，每關最後都有較高強的頭目把守，玩者
      必須一路前進打敗各關敵人，以救出被綁的師妹。每關出現的敵人和把關的頭目都有
      其弱點，必須針對其弱點才能輕鬆過關。整體而言，這款遊戲設計得還算不錯，玩家
      可以玩玩看。

密技：將人物帶到最左邊朝向右時，則左邊出現的敵人將不會攻擊，而呆呆的站在那邊。將
      右邊出現的敵人都清光後，再轉身清掉左邊的敵人，比較不會兩面受敵。

      X或Z = 跳躍
      跳＋腳＋方向鍵 = 跳踢
      跳＋拳＋腳 = 氣功（會失血）
      P = 暫停
      S = 音效開關
