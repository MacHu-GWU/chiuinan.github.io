名稱：逆襲 (Opposite Attack，中文版)
簡介：大宇1989年出品的射擊遊戲，採用2D捲軸方式進行，全部共4關。遊戲己方一被擊中便
      斃命，再加上敵人與子彈相相當密集，因此想要過關可說是非常困難。聲光效果已比
      不上現今的射擊遊戲，懷舊效果遠大於娛樂效果。



保護：無
限制：單色螢幕，DosBox要設定machine=hercules。
限制：需要換片，必須在A磁碟啟動。DosBox若要執行，必須使用原始DOS開機，否則會當機。

密技：遊戲中按Esc暫停後，按下5431351，再按Esc繼續遊戲，即可變成無敵狀態。無敵狀態
      下按S鍵則取消無敵。
操作：

Space = 開始遊戲
Z = 發射子彈
A = 啟用／切換副武器攻擊方向
Esc = 暫停／恢復
Q = 離開遊戲
