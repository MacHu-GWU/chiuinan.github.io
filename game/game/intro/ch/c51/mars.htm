名稱：終極戰士 (Mars 3D，中文版)
簡介：安峻1997年出品的第一人稱射擊遊戲，由智冠代理發行，為毀滅戰士DOOM的仿造遊戲
      之一 [待補]。
 
備註：本版為1.2版
保護：光碟
密技：遊戲中直接輸入下列字串：

j62u6 = 無敵注音按鍵 = 無敵
j3fu4 = 武器注音按鍵 = 武器全滿

操作：

上下鍵 = 前進/後退
左右鍵 = 左轉/右轉
,.或Alt+左右鍵 = 左移/右移
Alt+上鍵 = 180度轉身
右Shift+上下鍵 = 跑步
Space = 使用/開啟/交談
Ctrl = 射擊
A = 跳起/浮上
Z = 蹲下/下潛
Caps = 跑步開關
End = 飛行開關
Ins/Del = 飛行上升/下降
PgUp/PgDn = 往上看/往下看
Home = 平視
Tab = 地圖開關
+/- = 地圖放大縮小
F1 = 按鍵說明
F5 = 快速存檔
F6 = 快速讀檔
Esc = 系統選單
