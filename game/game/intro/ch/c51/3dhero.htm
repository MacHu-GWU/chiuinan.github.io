名稱：三國志蜀漢英雄傳 (中文版)
簡介：安峻1997年製作的第一人稱射擊遊戲，由智冠代理發行。遊戲的形態類似於毀滅戰士，
      只是主角角色換成三國蜀漢將領孔明、關羽、張飛、趙雲等人，砍殺的則是敵軍 [待補]。



保護：光碟
操作：可使用SETUP.EXE變更部份按鍵

上下 = 前進/後退
右Shift+上下 = 跑步
左右 = 轉身
Alt+上 = 立即180度轉身
Alt+左/, = 左平移
Alt+右/. = 右平移
A = 跳躍
Z = 攻擊
Ctrl = 施展法術
; = 顯示／變更為上一個法術
' = 顯示／變更為下一個法術
1～9、0 = 直接變更法術
Enter = 使用物品
[ = 顯示／變更為上一個物品
] = 顯示／變更為下一個物品
Space = 開門／交談
Tab = 地圖開關
+/- = 畫面／地圖縮放
CapsLock = 跑步開關
PgUp = 往上看
PgDn = 往下看
Home = 平視
F7 = 快速存檔
F8 = 快速讀檔
F9 = 求助畫面
Esc = 系統選單
