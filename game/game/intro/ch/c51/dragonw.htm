名稱：龍女 (Dragon Girl，中文版)
簡介：大宇1989年出品的射擊遊戲，捲用3D前進捲軸方式進行，全部共10關。故事描寫風魔
      偷走龍珠，打開地獄之門放出惡靈為禍人間，龍女的任務就是要消滅這邪惡的風魔。
      遊戲己方一被擊中便斃命，再加上3D不易判斷物體遠近進行閃躲，敵人與子彈也頗為
      密集，因此想要過關可說是難如登天。不過操控上還算頗為順暢，比較適合想要挑戰
      的玩家。



保護：無
限制：單色螢幕，DosBox要設定machine=hercules。
備註：正常版請執行PLAY.BAT，不死版請執行UPLAY.BAT
修改：DRAGON.EXE

1.不死
  FF 0E 93 00
  90 90 90 90
2.無限多隻
  FF 0E 51 00
  90 90 90 90

操作：

Space = 開始遊戲
X = 發射
Esc = 暫停
D = 重玩本關
Q = 結束離開
C = 設定
  F1 = 變更穩定遊戲速度
  F2 = 變更遊戲控制方式
  F3 = 變更電腦執行速度
  F4 = 武器發射速度
  F4 = 變更背景音效設定(PC喇叭)

說明：各過一關加一命，全部共十關。
