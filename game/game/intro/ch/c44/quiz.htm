名稱：美女綜藝大集合 (中文版)
簡介：全崴1993年出品的１８禁猜謎遊戲，玩者必須經歷自然科學、運動知識、藝文活動、
      五花八門、生理衛生、中外歷史、風土人情等七類問題，在每類問題中通過三次考驗，
      再猜拳打敗美女，便可看到一張美女圖。遊戲中提供有許多輔助猜謎與猜拳的物品，
      且每通過一次考驗，也有一些額外的獎品。至於所問的問題相當多種，有難也有簡單
      的，但要過關可非易事。
 
保護：數字密碼，破解碼：

密技：

1.美女脫衣圖
  在選擇測驗科目時，按下Ctrl鍵及輸入TERRY可看見全部過關後的所有美女不著衣衫。
2.增加Power
  在購買密技頁時，按下Ctrl鍵及輸入GOD，Power值會增加150點，但只能用一次。
3.助理小姐脫衣圖
  在助理小姐出現的問答題時，按下Ctrl鍵及輸入QUIZ，會出現助理小姐脫衣圖。
4.活動美女
  用正常方法贏得五根火柴後，出現美女圖時，按下Ctrl鍵及輸入BEAUTY會出現一個游標，
  在圖面上某一點按滑鼠左鍵，美女就會動起來。

操作：

滑鼠左鍵/Space = 確認／開始／下一頁
滑鼠右鍵/Enter = 打開／關閉秘技視窗
上下左右 = 移動游標
Esc = 離開／結束遊戲
M = 音樂開關
E = 音效開關
V = 語音開關

說明：

1.共有自然科學、運動知識、藝文活動、五花八門、生理衛生、中外歷史、風土人情7個單元。
2.每個單元分成3關，答題時間限1分30秒（在不使用秘技的情況下）：
  (1)第一關：必須答對10題才可過關。
  (2)第二關：必須答對13題才可過關。
  (3)第三關：必須答對16題才可過關。
3.答題過程可隨時按下滑鼠右鍵或Enter鍵，打開秘技視窗使用秘技。
4.若短時間內即回答正確答案，可額外獲得Power值。
5.獎金計算方式：
  (1)第一關
     答題獎金 = 答對題數x1000元
     時間獎金 = 過關剩餘秒數x100元
     連莊獎金 = 已(連續)參與過的單元數x5000元
  (2)第二關
     答題獎金 = 答對題數x2000元
     時間獎金 = 過關剩餘秒數x200元
     連莊獎金 = 已(連續)參與過的單元數x10000元
  (3)第三關
     答題獎金 = 答對題數x3000元
     時間獎金 = 過關剩餘秒數x500元
     連莊獎金 = 已(連續)參與過的單元數x20000元
6.同一單元闖過三關後，即進入火柴姑娘姑娘遊戲。
  (1)總共猜拳5次，每獲勝一次便可得到一根火柴。
  (2)愈多火柴，可看到的美女畫面愈高，範圍也愈大。
  (3)購買「好運火柴」或「全勝火柴」秘技後，必須進入秘技視窗加以使用才有效。

*** 拿獎品遊戲

獎品共分為五個等級（以字母A～H為代表）：

1.21世紀休閒組合（可加Power值）
2.高級精美贈品（可加Power值）
3.家庭實用禮品（可加Power值）
4.心跳100的驚喜（會扣Power值）
5.恐怖整人禮盒（會扣Power值）

一、彈球台

1.按下滑鼠左鍵或空白鍵再放開，依按住時間決定彈出力道。
2.凡是在A與H之間，某一個字母累積了兩顆彈珠，即會得到此字母所代表的獎品。
3.若玩到第8顆珠子才結束，可額外獲得150點Power值。若玩到第9顆珠子才結束，則可獲得
  400點Power值。

二、活人轉盤

1.先按一次滑鼠左鍵或空白鍵讓活人開始轉動。
2.再按一次滑鼠左鍵或空白鍵讓字母開始轉動。
3.全部停止後，活人頭頂指向的字母即為獎品。

三、立體連連看

1.將滑鼠游標移動到想要的起始位置，再按下滑鼠左鍵或空白鍵開始進行。
2.最終連到的字母即為獎品。

四、七彩瘋狂大摸彩

1.先按一次滑鼠左鍵或空白鍵開始轉動。
2.再按一次滑鼠左鍵或空白鍵停止轉動。
3.全部停止後，視出現的圖像決定獎品。若是得到21世紀休閒組合，則可欣賞該位美女圖。

五、賓果水果盤

1.逐一翻開左側蓋著的牌，依顏色放到右側水果中。
2.當某個水果顏色集滿後，即代表獲得該項獎品。
