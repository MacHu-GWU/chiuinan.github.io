名稱：魔境大冒險 (中文版)
簡介：傑克豆1995年出品的益智遊戲，和俄羅斯方塊類似。遊戲分成冒險、對戰、挑戰關卡
      等多種模式，亦可自行編輯關卡。冒險部份描寫公主被冰雪女王擄走，主角闖關解救
      出公主，其過程亦為挑戰關卡。本遊戲雖和俄羅斯方塊類似，但主要目的利用滾球除
      去所有已存在的動物，而非消去一列列的方塊，由於滾球有數目限制，因此如何找出
      正確的解法，需要動動腦筋並多方嘗試，所幸關卡與物品掉落的次序都是固定，且可
      無限次數嘗試。至於是否好玩，就見仁見智了。
 
保護：顏色密碼，破解碼：
      MU.EXE  (不用解壓)
        3A C2 75 04
        88 -- -- 00
操作：

一、單人（右方）

左右鍵 = 左右移
Space = 旋轉方塊
下鍵 = 加快掉下速度
Shift-Q = 跳回DOS

二、雙人

1.右方

左右鍵 = 左右移
Space/Ins = 旋轉方塊
下鍵 = 加快掉下速度
上鍵 = 使用寶物

2.左方

AS = 左右移
Ctrl = 旋轉方塊
Z = 加快掉下速度
W = 使用寶物
