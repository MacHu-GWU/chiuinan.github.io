名稱：超級天九VS野球拳 (中文版)
簡介：船塢資訊1990年出品的ADD 9賭博遊戲，由資訊週刊代理發行。除了傳統的ADD 9外，
      遊戲還加上了類似梭哈的BONUS規則，讓遊戲有更多的變化。再加上額外的野球拳遊戲、
      猜獎遊戲，其實娛樂性還算不錯。



備註：原版磁片染有tenbytes病毒，且壓縮前已感染，壓縮後再度感染，本版已加以清除。
保護：數字密碼，破解碼（密碼表）：
      KK3.PIC
        3B 46 F2 75 07  (任意輸入4個數字)
        -- -- -- -- 00
限制：只能在單色螢幕上執行，DosBox請設定machine=hercules。
修改：ADD9.DAT (先用unp解壓)

操作：

M = 補牌／起動
X = 開牌／得分
B = 賭倍
S = 賭小
K = 賭大
Esc = 跳回DOS／跳出野球拳遊戲

說明：

一、天九牌

1.玩家初始金錢為100。
2.螢幕下方為玩家的牌，上方四家為電腦控制對手的牌。
3.發完第一張牌後，玩家決定下注金額為1～9（若按M鍵表示為上次下注金額）。
4.開始發第二張牌與補牌，玩家可視四家對手的點數總和個位數來決定是否補牌。最大為9，
  最小為0。
5.電腦對手發牌或補牌時，若出現賭神牌，則其下注金額加倍。
6.得分若賭倍連贏三次，則可進入野拳球遊戲。
7.玩家若得到下列牌組，則無條件獲勝，並進行猜獎遊戲：
  A.同花順234：得分=下注金額*20倍
  B.三張9：得分=下注金額*20倍
  C.三張3：得分=下注金額*20倍
  D.三張同花點數9：得分=下注金額*10倍
  E.順子234：得分=下注金額*10倍
  F.同花順123：得分=下注金額*7倍
  G.三張同花順：得分=下注金額*4倍
  H.三條：得分=下注金額*4倍
  I.順子123：得分=下注金額*3倍
  J.三張順子：得分=下注金額*2倍
  K.三張同花：得分=下注金額*2倍
  J.兩張A9：得分=下注金額*2倍
8.金錢達到3000分，則算破台，遊戲結束。

二、野球拳

1.以剪刀、石頭、布，猜拳決勝負。
2.隨著我方贏的次數，美女會逐漸寬衣解帶。
3.必須贏美女五次才算過關，但我方只有三次失敗機會。

三、猜獎遊戲

1.右上角的牌，必須與螢幕上依次翻出的牌比點數大小。
2.猜贏金錢加10，平點加5，猜輸不扣錢。
